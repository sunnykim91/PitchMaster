시작하기

1가입하기

별도 회원가입 없이, 카카오 로그인 한 번이면 됩니다.

  1. pitch-master.app 접속
  2. 카카오 로그인 클릭
  3. 이름과 선호 포지션 입력 (연락처·생년월일은 나중에 해도 됩니다)
  4. 새 팀 만들기 또는 초대 링크로 기존 팀에 가입
가입 전에 먼저 둘러보고 싶다면 → 메인 페이지의 "30초 만에 데모 체험하기" 버튼을 눌러보세요.

2팀 만들기 & 팀원 초대

  1. 팀명 입력 + 축구/풋살 선택 → 팀 생성
  2. 대시보드에서 "카카오톡으로 초대" 버튼을 눌러 팀원에게 공유
  3. 팀원이 링크를 눌러 카카오 로그인하면 자동으로 팀에 합류됩니다
📋 팀원에게 보낼 초대 안내 (복사해서 카톡에 붙여넣기)
우리 팀 참석투표/회비관리 앱이에요!
아래 링크 눌러서 카카오 로그인하면 바로 가입돼요 👇

👉 https://www.pitch-master.app/team?code=[초대코드]

※ 초대코드는 대시보드에서 확인할 수 있어요
팀원은 어떻게 가입하나요?
  1. 회장/총무가 보낸 초대 링크를 클릭
  2. 카카오 로그인 (3초)
  3. 이름과 선호 포지션 입력하면 자동 가입

링크 하나면 바로 가입 — 앱 설치나 별도 회원가입이 필요 없습니다.

초대 코드 없이 팀 찾기
  1. 가입 후 팀 검색해서 가입을 선택
  2. 팀 이름을 검색하여 가입 신청
  3. 운영진이 승인하면 자동으로 팀에 합류됩니다

※ 회장이 설정 > 팀설정에서 "팀 검색 허용"을 켜야 검색에 노출됩니다.

경기 운영

3경기 운영 흐름

이 순서만 기억하면 됩니다.

📅 일정 등록 → 🗳️ 투표 → 📋 라인업 → 📋 출석 → ⚽ 기록 → 📤 공유

📅 경기 등록

경로: 하단 메뉴 "일정" → 우측 상단 [+] 버튼

1. 날짜/시간: 경기 날짜, 시작~종료 시간 설정
2. 장소: 구장 이름 입력
3. 상대팀: 상대팀 이름 (자체전이면 "자체전" 토글)
4. 경기 유형: 일반 경기 / 자체전 / 팀 일정(회식 등) 선택
5. 투표 마감: 자동으로 전날 오후 5시로 설정 (수정 가능)
6. 등록하면 팀원에게 자동 푸시 알림 발송
팀 일정(EVENT)을 선택하면 회식, 워크샵 같은 비경기 일정도 등록하고 투표받을 수 있습니다.

✏️ 경기 수정/삭제

경로: 경기 상세 → 정보 탭 → 우측 상단 연필 아이콘

· 날짜, 시간, 장소, 상대팀, 투표 마감일 수정 가능
· 하단에 "경기 삭제" 버튼 (모든 기록 함께 삭제, 복구 불가)
· 전적 반영 토글: 시즌 통계에 포함할지 ON/OFF 선택

권한: 운영진 이상만 수정/삭제 가능

역할할 수 있는 것

4참석 투표

매주 "이번 주 누가 와요?" 물어볼 필요 없이, 자동 집계됩니다.

경기 등록 → 자동 알림 → 참석/불참/미정 → 실시간 집계
투표 방법 (회원):
1. 푸시 알림 또는 카톡 공유 링크 클릭
2. 경기 상세 → 투표 탭
3. [참석] [미정] [불참] 버튼 클릭

운영진 기능:
· 대리 투표: 카톡으로 참석 의사를 전달받은 회원 대신 투표
· 미투표자 확인: 투표 안 한 회원을 한눈에 확인
· 투표 마감: 마감 시간 설정 → 전날 리마인더 자동 발송
· 투표 링크 공유: 카카오톡으로 투표 링크 바로 전송
경기 목록에서도 바로 투표할 수 있습니다. 카드 하단의 [참석] [미정] [불참] 버튼을 눌러보세요.

5회비 관리

통장 내역을 올리면 자동으로 정리됩니다. 5개 탭으로 회비를 완벽하게 관리하세요.

회비 설정 → 통장 내역 올리기 → 자동 인식 → 납부 현황 확인

📊 입출금 탭

· 월별 입출금 내역을 한눈에 확인
· 수기 입력: 유형(입금/출금) · 금액 · 날짜 · 시간 · 결제수단(출금만) 직접 등록
· 내용에 팀원 이름이 포함되면 납부 기록 자동 연결
· 잔고 반영 여부 선택 가능 (체크박스)
· 수정/삭제는 아이콘 버튼으로 간편하게

✅ 납부현황 탭 운영진

· 미납 · 면제 · 납부 완료 그룹별 접기/펼치기
· 납부 자동 확인: 입금 내역에서 회비 금액과 이름을 매칭하여 자동 처리
· 미납 알림: 미납 회원에게 푸시 알림 일괄 전송
· 장기 미납자 자동 감지 (2개월 이상 연속 미납)
· 회원별 납부 상태를 수동으로 변경 가능 (납부/미납/면제)

📤 업로드 탭 운영진

· AI OCR: 통장 거래내역 캡쳐를 올리면 AI가 자동 인식 (이름 · 금액 · 날짜 · 시간 · 메모). 인식률이 낮은 캡쳐는 Claude Vision으로 자동 폴백.
· 엑셀 업로드: 은행 앱에서 다운로드한 파일을 그대로 올리면 됩니다
· 중복 내역 자동 감지 및 제외
· 인식 결과를 편집한 후 일괄 저장
· 잔고 자동 반영 (기존 잔고보다 최신일 때만)
지원 은행: 카카오뱅크, 토스뱅크, 국민, 신한, 우리, 하나, 농협
💰 잔고 안내
통장 잔고는 스크린샷/엑셀 업로드 시에만 갱신됩니다.

Q. 왜 수기 입력/삭제하면 잔고가 안 바뀌나요?
수기로 입력하거나 삭제하는 건 앱에서 기록을 정리하는 거지, 실제 통장에서 돈이 들어오고 나간 게 아닙니다. 만약 이걸 잔고에 반영하면 잘못 입력한 걸 삭제하거나, 테스트로 넣었다 지울 때마다 잔고가 틀어져서 실제 통장 잔고와 안 맞게 됩니다.
그래서 잔고는 실제 은행 데이터(스크린샷/엑셀)를 올릴 때만 맞춰집니다.

잔고가 안 맞으면? → 최신 통장 내역 캡쳐를 다시 올려주세요!

⚙️ 설정 탭 운영진

· 납부 기준일: 매월 N일 기준으로 회비 기간 자동 계산
· 회비 유형: 일반 · 학생 · 신입 등 유형별 금액 설정, 인라인 편집
· 벌금 규칙: 지각 · 불참 · 미투표 시 자동 벌금 부과 규칙 등록
· 회원 회비 상태: 면제 · 휴회 · 부상 등 기간 설정 (아래 상세)

🏥 회원 회비 상태 (면제/휴회/부상)

장기 면제가 필요한 회원을 등록하면 매월 자동으로 면제 처리됩니다.

· 면제: 운영진, 특별 면제 등 (예: "운영진 면제")
· 휴회: 장기 휴식 (예: "해외 출장 3개월")
· 부상: 부상으로 인한 면제 (예: "무릎 수술 회복")

사용법:
1. 설정 탭 → 하단 "회원 회비 상태" → [추가]
2. 회원 선택, 상태(면제/휴회/부상), 사유, 시작일~종료일 입력
3. 등록 완료 → 해당 기간 동안 납부현황에서 자동 면제 처리
4. 종료일이 지나면 자동으로 일반 상태 복귀
5. [종료] 버튼으로 조기 종료 가능, 이력은 보존됨
이미 "납부 완료"로 처리된 회원은 면제로 덮어쓰지 않습니다.

💰 벌금 탭

경기 완료 시 설정된 규칙에 따라 자동으로 벌금이 부과됩니다.

벌금 규칙 등록:
1. 회비 > 설정 탭 > 하단 "벌금 규칙" → [추가]
2. 규칙 이름 (예: "지각비"), 트리거 (지각/불참/미투표/기타), 금액 입력
3. 저장 → 이후 경기 완료 시 자동 적용

벌금 부과 흐름:
1. 경기 당일 → 출석 탭에서 참석/지각/불참 체크
2. 경기 완료 처리 시 → 지각/불참 회원에게 벌금 자동 부과
3. 회비 > 벌금 탭에서 회원별 벌금 현황 확인 및 납부 처리

5-1출석 체크

경기별로 참석/지각/불참을 기록합니다.

경기 등록 → 참석 투표 → 출석 체크 → 벌금 자동 부과
사용법:
1. 경기 상세 → 출석 탭 선택
2. 참석 투표한 회원 목록이 표시됨
3. 각 회원의 [참석] [지각] [불참] 버튼 클릭
4. [전원 참석 처리] 버튼으로 일괄 처리 가능
5. 하단에 출석 현황 요약 (참석 N명, 지각 N명, 불참 N명) 표시

권한: 운영진 이상만 출석 체크 가능
벌금 연동: 벌금 규칙에 "지각" 또는 "불참"이 설정되어 있으면 경기 완료 시 자동 부과

5-2유니폼 관리

홈/원정/써드 유니폼을 설정하고 경기별로 선택합니다.

유니폼 설정:
1. 설정 > 팀 설정 → 하단 "유니폼" 섹션
2. 홈/원정 탭에서 메인 색상, 서브 색상, 패턴 선택
3. 패턴: 단색 · 세로 스트라이프 · 가로 스트라이프 · 대각 스트라이프
4. 써드 유니폼: [+ 써드 추가] 버튼으로 추가 (선택사항)
5. 저장 → 대시보드, 경기 목록에 미니 져지로 표시

경기별 유니폼 선택:
1. 경기 상세 → 정보 탭
2. "유니폼" 영역에서 [홈] [원정] [써드] 버튼 선택
3. 선택한 유니폼이 경기 목록, 대시보드, 전술판에 반영

6AI 전술 편성 (전술판)

두 가지 편성 모드로 전술판을 채웁니다. 공정한 쿼터 분배는 기본.

참석 확정 → 편성 방식 선택 → 수정 → 공유
편성 방식 2가지:

① 규칙 기반으로 빠르게 — 팀 기본 포메이션에 감독지정/선호 포지션 우선으로 즉시 배치. 비용 0·지연 0.
② AI 풀 플랜 — 우리 팀 승률 높은 포메이션·참석자 실력·상대 이력을 종합해 쿼터별 다른 포메이션까지 AI가 설계. 편성이 끝나면 감독 작전 브리핑(공격 루트·수비 원칙·고비 쿼터)이 함께 생성됩니다.

사용법:
1. 경기 상세 → 전술 탭 선택
2. [자동 편성 실행] 또는 [AI 풀 플랜 받기] 버튼 클릭
3. 쿼터별 탭(1Q, 2Q, 3Q, 4Q)으로 전환하며 확인
4. 선수를 드래그하여 위치 수정 가능
5. 우측 상단 공유 아이콘으로 전술판 이미지 저장/카톡 공유

쿼터 분배 원칙: 감독지정 포지션 우선 → 없으면 선호 포지션. 모두 참석자 간 공정 분배. GK는 1명이면 풀 출전, 여럿이면 로테이션.

AI 이용 한도: AI 풀 플랜은 경기당 2회(최초+재생성) + 팀당 월 10회. 빠른 편성 뒤 독립 호출하는 AI 코치는 경기당 3회.
전술판은 PC 브라우저에서도 똑같이 동작합니다. 경기 전날 밤엔 큰 화면으로 차분히, 현장에선 폰으로 즉시 수정하세요.

7경기 기록 & 통계

골, 어시스트, MVP를 경기 중에 바로 기록하고, 시즌 통계를 자동으로 집계합니다.

⚽ 골/어시스트 기록

경로: 경기 상세 → 기록 탭

1. [+ 득점] 버튼 → 즉시 등록 (득점자·쿼터·어시스트 나중에 수정 가능)
2. [+ 실점] 버튼 → 상대팀 득점 즉시 기록
3. 등록된 골 카드 터치 → 득점자 · 어시스트 · 쿼터 · 골 유형 수정
   · 쿼터: 1Q~4Q 또는 모름 선택 가능
   · 골 유형: 일반 · PK · FK · 헤딩 · 자책골
4. 스코어보드에 실시간 반영
5. 수정/삭제 가능 (해당 골 카드 우측 버튼)

권한: 모든 회원이 기록 가능 (팀 설정에서 운영진 전용으로 변경 가능)

🏆 MVP 투표

경로: 경기 상세 → 기록 탭 하단

1. 참석한 회원 중 MVP에 투표
2. 1인 1표, 변경 가능
3. 가장 많은 표를 받은 선수가 MVP 뱃지 획득
4. 시즌 통계에 MVP 횟수 자동 집계

📓 경기 일지 & 자동 경기 후기

경로: 경기 상세 → 일지 탭

1. 자동 경기 후기: 경기가 COMPLETED 상태면 득점·MVP·참석 데이터로 한 문장 후기가 자동 생성돼 공유 카드 상단에 표시됩니다. 재생성은 경기당 1회.
2. 날씨: 맑음/흐림/비/눈/바람 선택
3. 컨디션: 1~5단계 (나쁨→좋음)
4. 메모: 경기 후기, 특이사항 자유 입력
5. 사진: 경기 사진 업로드 (3열 그리드, 확대 보기)
6. 공유: 결과 카드를 카카오톡으로 공유 또는 클립보드 복사

경로: 하단 메뉴 "기록"

⚽ 자체전 (A팀 vs B팀)
· 경기 등록 시 "자체전" 토글을 켜면 활성화
· 랜덤 배정 또는 수동으로 A/B팀을 나눌 수 있습니다
· A팀 vs B팀 별도 스코어보드와 팀별 골/어시스트 기록

· "통계 반영"을 체크하면 시즌 통계에 포함됩니다

· 대시보드와 경기 목록에서 유니폼 색상이 자동으로 표시됩니다

9홈 화면에 추가하는 방법

홈 화면에 추가하면 앱처럼 바로 접속할 수 있습니다.

🍎 아이폰 (Safari)
  1. Safari로 pitch-master.app 접속
  2. 하단 메뉴바에서 공유 아이콘 (⬆︎) 탭
  3. "홈 화면에 추가" 선택
  4. 우측 상단 "추가" 탭
🤖 안드로이드 (Chrome, 삼성 인터넷 등)
  1. 브라우저로 pitch-master.app 접속
  2. 하단에 뜨는 "홈 화면에 추가" 배너를 탭
  3. 또는 앱 안의 더보기 > 홈 화면에 추가 버튼 탭
홈 화면에 추가하면 알림(푸시)도 받을 수 있습니다. 설정 > 알림에서 켜주세요.

10FAQ

앱 설치가 필요한가요?
아닙니다. 링크만 누르면 바로 사용할 수 있습니다. 홈 화면에 추가하면 일반 앱과 똑같이 사용할 수 있고, 알림도 받을 수 있습니다.
팀원이 앱을 깔아야 하나요?
아니요. 초대 링크를 누르면 바로 카카오 로그인 → 가입 완료입니다. 앱 설치를 부탁할 필요가 없어서 팀원 가입률이 높습니다.
여러 팀에 가입할 수 있나요?
네, 한 계정으로 여러 팀에 소속될 수 있습니다. 좌측 메뉴에서 팀을 전환하면 됩니다.
카카오 인앱 브라우저에서 안 되는 게 있나요?
카카오톡 안에서 링크를 열면 알림, 홈 화면 추가가 제한됩니다. 우측 상단 ··· → "외부 브라우저에서 열기"를 누르면 모든 기능을 사용할 수 있습니다.
이전 시즌 데이터를 옮길 수 있나요?
네. 엑셀이나 스프레드시트로 기존 데이터(경기 결과, 회비 내역, 멤버 목록)를 보내주시면 일괄 등록 도와드립니다. 오픈채팅방에서 말씀해주세요.
회비 납부 기준일을 변경할 수 있나요?
네. 회비 > 설정 탭에서 납부 기준일을 변경할 수 있습니다. 예를 들어 기준일을 25일로 설정하면 매월 25일부터 다음달 24일까지를 한 달로 계산합니다.
완료된 경기의 참석자를 수정할 수 있나요?
네, 운영진 이상은 완료된 경기에서도 투표 탭에서 참석자를 추가/수정할 수 있습니다.
회원을 장기 면제/휴회 처리하려면?
회비 > 설정 탭 > 하단 "회원 회비 상태"에서 [추가]를 눌러 면제/휴회/부상 상태를 등록하세요. 시작일~종료일을 설정하면 해당 기간 동안 매월 자동으로 면제 처리됩니다. 종료일을 비워두면 무기한입니다.
벌금 규칙은 어떻게 설정하나요?
회비 > 설정 탭 > "벌금 규칙"에서 [추가]를 눌러 규칙 이름(예: 지각비), 트리거(지각/불참/미투표/기타), 금액을 설정합니다. 경기가 완료되면 해당 규칙에 따라 자동으로 벌금이 부과됩니다.
유니폼은 어떻게 설정하나요?
설정 > 팀 설정 > 유니폼에서 홈/원정/써드 유니폼의 메인 색상, 서브 색상, 패턴(단색/세로줄/가로줄/대각선)을 설정합니다. 경기 정보 탭에서 어떤 유니폼을 입을지 선택할 수 있습니다.
무료인가요?
현재 모든 기능을 무료로 사용할 수 있습니다. 추후 운영 정책에 따라 유료 요금제가 도입될 수 있으며, 변경 시 사전에 안내드립니다.
