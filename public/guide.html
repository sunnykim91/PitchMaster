시작하기

1가입하기

별도 회원가입 없이, 카카오 로그인 한 번이면 됩니다.

  1. pitch-master.app 접속
  2. 카카오 로그인 클릭
  3. 이름과 선호 포지션 입력 (연락처·생년월일은 나중에 해도 됩니다)
  4. 새 팀 만들기 또는 초대 링크로 기존 팀에 가입
가입 전에 둘러보고 싶다면 메인 페이지의 "30초 만에 데모 체험하기" 버튼을 눌러보세요.

2팀 만들기 · 팀원 초대

  1. 팀명 입력 + 축구/풋살 선택 → 팀 생성
  2. 대시보드에서 "카카오톡으로 초대" 버튼을 눌러 팀원에게 공유
  3. 팀원이 링크를 눌러 카카오 로그인하면 자동으로 팀에 합류됩니다
팀원에게 보낼 초대 안내 (복사해서 카톡에 붙여넣기)
우리 팀 참석투표/회비관리 앱이에요!
아래 링크 눌러서 카카오 로그인하면 바로 가입돼요

https://www.pitch-master.app/team?code=[초대코드]

※ 초대코드는 대시보드에서 확인할 수 있어요
팀원은 어떻게 가입하나요?
  1. 회장/총무가 보낸 초대 링크 클릭
  2. 카카오 로그인 (3초)
  3. 이름·선호 포지션 입력하면 자동 가입
링크 하나면 바로 가입 — 앱 설치나 별도 회원가입 불필요.
초대 코드 없이 팀 찾기
  1. 가입 후 팀 검색해서 가입 선택
  2. 팀 이름 검색 → 가입 신청
  3. 운영진이 승인하면 자동 합류
※ 회장이 설정 → 팀 설정에서 "팀 검색 허용"을 켜야 검색에 노출됩니다.
경기 운영

3경기 운영 흐름

이 순서만 기억하면 됩니다.

일정 등록→ 투표→ 라인업→ 출석→ 기록→ 공유

경기 등록

하단 메뉴 "일정" → 우측 상단 [+] 버튼

  1. 날짜·시간: 경기 날짜, 시작~종료 시간
  2. 장소: 구장 이름
  3. 상대팀: 상대팀 이름 (자체전이면 토글)
  4. 경기 유형: 일반 경기 · 자체전 · 팀 일정(회식 등)
  5. 투표 마감: 자동으로 전날 오후 5시 (수정 가능)
  6. 등록하면 팀원에게 자동 푸시 알림 발송
팀 일정(EVENT)을 선택하면 회식·워크샵 같은 비경기 일정도 등록하고 투표받을 수 있습니다.

경기 수정 · 삭제

경기 상세 → 정보 탭 → 우측 상단 연필 아이콘. 운영진

  • 날짜·시간·장소·상대팀·투표 마감일 수정
  • 하단 "경기 삭제" 버튼 — 모든 기록 함께 삭제, 복구 불가
  • 전적 반영 토글: 시즌 통계 포함 여부 ON/OFF

4참석 투표

매주 "이번 주 누가 와요?" 물어볼 필요 없이, 자동 집계됩니다.

경기 등록→ 자동 알림→ 참석/불참/미정→ 실시간 집계

회원이 투표하는 법

  1. 푸시 알림 또는 카톡 공유 링크 클릭
  2. 경기 상세 → 투표 탭
  3. [참석] [미정] [불참] 버튼 클릭

운영진 기능 운영진

  • 대리 투표: 카톡으로 의사 전달받은 회원 대신 투표
  • 미투표자 확인: 투표 안 한 회원을 한눈에
  • 투표 마감: 마감 시간 설정 → 전날 리마인더 자동 발송
  • 투표 링크 공유: 카카오톡으로 투표 링크 바로 전송
경기 목록에서도 카드 하단의 [참석] [미정] [불참] 버튼으로 바로 투표할 수 있습니다.

5출석 체크

경기별로 참석·지각·불참을 기록합니다. 운영진

경기 등록→ 참석 투표→ 출석 체크→ 벌금 자동 부과
  1. 경기 상세 → 출석 탭
  2. 참석 투표한 회원 목록이 표시됨
  3. 각 회원의 [참석] [지각] [불참] 버튼 클릭
  4. [전원 참석 처리] 버튼으로 일괄 처리 가능
벌금 규칙에 "지각" 또는 "불참"이 설정되어 있으면 경기 완료 시 자동 부과됩니다.

6전술 편성

두 가지 편성 모드 · 부심·주심·촬영 슬롯 · 본인 역할 가이드까지 한 화면에서.

편성 방식 두 가지

  1. 규칙 기반으로 빠르게 — 팀 기본 포메이션에 감독지정·선호 포지션 우선 즉시 배치. 비용 0·지연 0.
  2. AI 풀 플랜 — 우리 팀 승률 높은 포메이션·참석자 실력·상대 이력을 종합해 쿼터별 다른 포메이션까지 설계. 편성 후 감독 작전 브리핑(공격 루트·수비 원칙·고비 쿼터)이 함께 생성됩니다. AI 섹션 참고.

사용 흐름

  1. 경기 상세 → 전술 탭
  2. [자동 편성 실행] 또는 [AI 풀 플랜 받기] 클릭
  3. 쿼터별 탭(1Q · 2Q · 3Q · 4Q)으로 전환하며 확인
  4. 선수를 드래그하여 위치 수정
  5. 우측 상단 공유 아이콘으로 전술판 이미지 저장·카톡 공유

쿼터 분배 원칙

  • 감독지정 포지션 우선 → 없으면 선호 포지션
  • 모든 참석자 간 공정 분배 (출전 쿼터 수 균등)
  • GK는 1명이면 풀 출전, 여럿이면 로테이션
  • 용병은 정규 슬롯이 다 채워진 뒤 자동 보조 배치

주심 · 부심 · 촬영 슬롯

쿼터별 정규 슬롯 외에도 경기 운영 인력을 슬롯으로 배치할 수 있습니다.

  • 주심 1명 · sky 색상
  • 부심 2명 · emerald 색상
  • 촬영 1명 · violet 색상

전술판 하단 카드에서 드래그하여 배치합니다. 편성 완료 판단은 정규 슬롯 기준 — 메타 슬롯은 영향 없음.

역할 가이드 (쿼터별 본인 포지션)

편성이 완료되면 전술판 하단에 "내 역할 가이드" 카드가 자동 표시됩니다.

  • 쿼터별 본인 포지션의 주요 역할 · 주의점 · 연계 포지션
  • 같은 (포메이션, 역할)이면 비연속 쿼터도 한 카드 ("2·4쿼터 RCB")
  • 포메이션이 바뀌면 별도 카드로 분리
  • 운영진은 드롭다운으로 다른 선수 시점도 확인 가능 (용병 제외)
지원 범위: 축구 11인제 10개 포메이션. 풋살·8/9/10인제는 미지원.
전술판은 PC 브라우저에서도 똑같이 동작합니다. 경기 전날 밤엔 큰 화면으로 차분히, 현장에선 폰으로 즉시 수정하세요.

전술 영상 — 우리 팀만의 빌드업·수비 만들기 운영진

역할 가이드 카드에 "팀 움직임 보기" 토글을 켜면 11명이 시퀀스로 움직이는 영상이 재생됩니다. 표준 영상이 자동 노출되지만, 운영진이 직접 우리 팀만의 영상을 만들 수도 있어요.

만드는 동선
햄버거 메뉴 → 설정 → 팀 설정 탭 → "감독의 전술노트 — 공격·수비 영상 만들기" 카드 → 포메이션 선택 후 새 영상 만들기
  • 11명 점·공 직접 드래그로 위치 조정. 장면(공격/수비) 안에 컷을 여러 개 두면 이어 재생되며 영상이 됩니다.
  • 컷마다 설명 한 줄(예: "GK → 좌측 센터백") 입력 가능
  • 대표 영상으로 지정하면 그 포메이션의 경기 역할 가이드 "팀 움직임 보기"에 우리 팀 영상이 자동 노출 (기본 4-2-3-1, 풋살 5인제 등 모두 가능)
  • 복제: 4-2-3-1 영상을 4-3-3 같은 다른 포메이션으로 복사해 시작점으로 사용 가능
  • 버전 관리: 같은 포메이션에 v1/v2/v3 여러 영상 보관 후 대표로 전환 가능
처음엔 4-2-3-1 표준 풀 시퀀스(GK → 센터백 → 미드 → 윙백 → 크로스 → ST 마무리)가 7컷 영상으로 들어있어요. 그대로 써도 되고, 우리 팀 스타일로 수정해도 OK.

7경기 기록 · 통계

골·어시스트·MVP를 경기 중에 바로 기록하고, 시즌 통계를 자동 집계합니다.

골 · 어시스트 기록

경기 상세 → 기록 탭

  1. [+ 득점] 버튼 → 즉시 등록 (득점자·쿼터·어시스트 나중에 수정 가능)
  2. [+ 실점] 버튼 → 상대팀 득점 즉시 기록
  3. 등록된 골 카드 터치 → 득점자 · 어시스트 · 쿼터 · 골 유형 수정
    • 쿼터: 1Q~4Q 또는 "모름" 선택 가능
    • 골 유형: 일반 · PK · FK · 헤딩 · 자책골
  4. 스코어보드에 실시간 반영, 수정·삭제 가능
기본은 모든 회원이 기록 가능. 팀 설정에서 운영진 전용으로 변경할 수 있습니다.

MVP 투표 · 확정 정책

경기 상세 → 기록 탭 하단. 1인 1표, 변경 가능. 가장 많은 표를 받은 선수가 MVP 뱃지를 획득하고 시즌 통계에 자동 집계됩니다.

공식 확정 조건 (둘 중 하나)
① 운영진 1명 이상 투표 → 즉시 확정 (투표율 무관)
② 일반 팀원 투표가 실제 참석자의 70% 이상 → 최다득표자 자동 확정

70% 미달이면 투표 카드 하단에 안내 문구가 자동 노출되며, 운영진이 직접 MVP를 지정할 수 있습니다. 운영진이 직접 지정한 경우도 즉시 확정.

팀 설정에서 "운영진만 MVP 투표"를 켜면 일반 팀원 투표가 막힙니다.
왜 70%? 한두 명만 투표한 결과로 MVP가 정해지면 공정성이 떨어지기 때문입니다.

경기 일지 · 자동 경기 후기

경기 상세 → 일지 탭

  • 자동 경기 후기: 경기 COMPLETED 시 득점·MVP·참석 데이터로 한 문장 후기 자동 생성. 재생성은 경기당 1회. AI 섹션 참고.
  • 날씨: 맑음·흐림·비·눈·바람
  • 컨디션: 1~5단계
  • 메모: 경기 후기·특이사항 자유 입력
  • 사진: 경기 사진 업로드 (3열 그리드)
  • 공유: 결과 카드를 카카오톡 공유 또는 클립보드 복사

시즌 통계

하단 메뉴 "기록"

  • 시즌별 골·어시스트·MVP·출석률 자동 집계
  • 선수별 레이더 차트로 강점·약점 확인
  • 시즌 랭킹: 득점왕·어시스트왕·MVP왕·출석왕
  • 개인 프로필에서 본인의 시즌별 상세 기록 확인
자체전 (A팀 vs B팀)
  • 경기 등록 시 "자체전" 토글을 켜면 활성화
  • 랜덤 배정 또는 수동으로 A/B팀을 나눌 수 있음
  • A팀 vs B팀 별도 스코어보드와 팀별 기록
  • 전술판에서 A/B팀 탭 분리
  • "통계 반영"을 체크하면 시즌 통계에 포함
회비

8회비 관리

통장 내역을 올리면 자동으로 정리됩니다. 5개 탭으로 회비를 완벽하게 관리하세요.

회비 설정→ 통장 내역 올리기→ 자동 인식→ 납부 현황 확인

입출금 탭

  • 월별 입출금 내역을 한눈에 확인
  • 수기 입력: 유형·금액·날짜·시간·결제수단 직접 등록
  • 내용에 팀원 이름이 포함되면 납부 기록 자동 연결
  • 잔고 반영 여부 선택 (체크박스)

납부현황 탭 운영진

  • 미납 · 면제 · 납부 완료 그룹별 접기/펼치기
  • 납부 자동 확인: 입금 내역에서 회비 금액·이름 매칭
  • 미납 알림 일괄 전송, 장기 미납자 자동 감지(2개월 이상)
  • 회원별 납부 상태 수동 변경 가능 (납부·미납·면제)
  • [선납 등록] 버튼 → 9. 회비 선납

업로드 탭 운영진

  • AI OCR: 통장 캡쳐 → 이름·금액·날짜·시간·메모 자동 인식. AI 섹션 참고.
  • 엑셀 업로드: 은행 앱에서 다운로드한 파일 그대로 업로드
  • 중복 내역 자동 감지·제외, 인식 결과 편집 후 일괄 저장
  • 잔고 자동 반영 (기존 잔고보다 최신일 때만)
지원 은행: 카카오뱅크 · 토스뱅크 · 국민 · 신한 · 우리 · 하나 · 농협
잔고가 안 맞아요
통장 잔고는 스크린샷·엑셀 업로드 시에만 갱신됩니다. 수기 입력·삭제는 앱 기록 정리일 뿐 실제 통장에서 돈이 움직인 게 아니기 때문에 잔고에 반영하지 않습니다. 잔고가 안 맞으면 최신 통장 내역 캡쳐를 다시 올려주세요.

설정 탭 운영진

  • 납부 기준일: 매월 N일 기준으로 회비 기간 자동 계산
  • 회비 유형: 일반·학생·신입 등 유형별 금액 인라인 편집
  • 벌금 규칙: 지각·불참·미투표 시 자동 부과 규칙
  • 회원 회비 상태: 면제·휴회·부상 기간 설정

회원 회비 상태 (면제·휴회·부상)

장기 면제가 필요한 회원을 등록하면 매월 자동으로 면제 처리됩니다.

  • 면제: 운영진·특별 면제 (예: "운영진 면제")
  • 휴회: 장기 휴식 (예: "해외 출장 3개월")
  • 부상: 부상으로 인한 면제

설정 탭 → 하단 "회원 회비 상태" → [추가] → 회원·상태·사유·시작일~종료일 입력. 종료일이 지나면 자동으로 일반 상태 복귀, [종료] 버튼으로 조기 종료 가능 (이력 보존).

이미 "납부 완료"로 처리된 회원은 면제로 덮어쓰지 않습니다.

벌금 탭

경기 완료 시 설정된 규칙에 따라 자동 부과됩니다.

  1. 회비 → 설정 탭 → 하단 "벌금 규칙" → [추가]
  2. 규칙 이름(예: "지각비") · 트리거 · 금액 입력
  3. 경기 완료 → 지각·불참 회원에게 자동 부과
  4. 회비 → 벌금 탭에서 회원별 벌금 현황 확인·납부 처리

9회비 선납

6개월·1년치 회비를 미리 받고, 자동으로 매월 납부 처리합니다.

회비 → 납부현황 탭 → 우측 상단 [선납 등록] 운영진

  1. 회원 선택
  2. 선납 기간 선택 (3개월 · 6개월 · 12개월)
  3. 시작 월 선택
  4. 등록 완료 → 해당 기간의 매월이 자동으로 "납부 완료"로 표시
  • 매달 송금 부탁 안 해도 됨
  • 통장 입출금은 한 번에 처리
  • 휴회·면제와 충돌 시 자동 정합성 체크 (이미 납부된 월은 덮어쓰지 않음)
선납은 결제 인프라 없이 동작합니다 — "이번 분기 회비 한 번에 받기" 같은 카톡 송금 후 운영진이 등록하는 흐름입니다.

10월별 결산

매월 경기 결과·득점왕·MVP·회비 흐름을 한 페이지에서 확인합니다.

회비 → 우측 상단 "월별 결산" 버튼

  • 그 달 치른 경기 수 · 승무패
  • 득점왕 · 도움왕 · MVP왕 자동 집계
  • 그 달의 회비 입금/지출 합계 · 미납 회원 목록
  • 출석률 상위 회원
  • 카드 하나로 카카오톡 공유 — 매달 1일 단톡방에 깔끔히 전달
"이번 달 우리 팀 어땠지?"를 카톡에서 정리하던 총무를 위해 만들었습니다.

각 기능 자세히

1. AI 풀 플랜 — 우리 팀 승률 높은 포메이션·참석자 실력·상대팀 이력을 종합해 쿼터별 다른 포메이션까지 설계합니다. 편성 직후 감독 작전 브리핑(공격 루트·수비 원칙·고비 쿼터·체력 부담 선수)이 함께 생성됩니다.

2. AI 코치 분석 — 규칙 기반으로 빠르게 편성한 뒤, 별도로 코치 분석만 받고 싶을 때. 풀 플랜과 동일 품질의 작전 브리핑 + 선수 로테이션 분석.

3. 자동 경기 후기 — 득점·MVP·참석 데이터로 한 문장 후기가 자동 생성돼 공유 카드 상단에 표시됩니다. 카톡으로 결과 공유할 때 깔끔한 한 줄.

4. 선수 시그니처 — 선수의 시즌 누적 통계(골·어시·MVP·주포지션)를 바탕으로 AI가 한 줄 별명을 지어줍니다 (예: "우리 팀 황소", "왼발의 마술사"). 캐시되어 팀원 모두에게 노출.

5. 회비 OCR — Claude Vision이 통장 캡쳐를 분석해 이름·금액·날짜·시간·메모를 자동 추출합니다. 결과를 편집한 후 일괄 저장. 지원 은행: 카카오뱅크·토스뱅크·국민·신한·우리·하나·농협.

실패한 호출이나 룰 폴백은 한도에서 차감되지 않습니다.

FAQ

14자주 묻는 질문

앱 설치가 필요한가요?
아닙니다. 링크만 누르면 바로 사용할 수 있습니다. 홈 화면에 추가하면 일반 앱과 똑같이 사용할 수 있고, 알림도 받을 수 있습니다.
팀원이 앱을 깔아야 하나요?
아니요. 초대 링크를 누르면 바로 카카오 로그인 → 가입 완료입니다. 앱 설치를 부탁할 필요가 없어서 팀원 가입률이 높습니다.
여러 팀에 가입할 수 있나요?
네, 한 계정으로 여러 팀에 소속될 수 있습니다. 좌측 메뉴에서 팀을 전환하면 됩니다.
카카오 인앱 브라우저에서 안 되는 게 있나요?
카카오톡 안에서 링크를 열면 알림·홈 화면 추가가 제한됩니다. 우측 상단 ··· → "외부 브라우저에서 열기"를 누르면 모든 기능을 사용할 수 있습니다.
이전 시즌 데이터를 옮길 수 있나요?
네. 엑셀이나 스프레드시트로 기존 데이터(경기 결과·회비 내역·멤버 목록)를 보내주시면 일괄 등록 도와드립니다. 오픈채팅방에서 말씀해주세요.
회비 납부 기준일을 변경할 수 있나요?
네. 회비 → 설정 탭에서 납부 기준일을 변경할 수 있습니다. 기준일을 25일로 설정하면 매월 25일부터 다음달 24일까지를 한 달로 계산합니다.
회비 선납은 어떻게 등록하나요?
회비 → 납부현황 탭 → 우측 상단 [선납 등록] 버튼을 누르면 회원·기간(3·6·12개월)·시작 월을 골라 등록할 수 있습니다. 등록 즉시 해당 기간의 매월이 자동으로 "납부 완료"로 표시됩니다. 자세한 내용은 9. 회비 선납 참고.
완료된 경기의 참석자를 수정할 수 있나요?
네, 운영진 이상은 완료된 경기에서도 투표 탭에서 참석자를 추가·수정할 수 있습니다.
회원을 장기 면제·휴회 처리하려면?
회비 → 설정 탭 → 하단 "회원 회비 상태"에서 [추가]를 눌러 면제·휴회·부상 상태를 등록하세요. 시작일~종료일을 설정하면 해당 기간 동안 매월 자동으로 면제 처리됩니다. 종료일을 비워두면 무기한입니다.
벌금 규칙은 어떻게 설정하나요?
회비 → 설정 탭 → "벌금 규칙"에서 [추가]를 눌러 규칙 이름·트리거(지각·불참·미투표·기타)·금액을 설정합니다. 경기가 완료되면 해당 규칙에 따라 자동으로 벌금이 부과됩니다.
유니폼은 어떻게 설정하나요?
설정 → 팀 설정 → 유니폼에서 홈·원정·써드 유니폼의 메인 색상·서브 색상·패턴을 설정합니다. 경기 정보 탭에서 어떤 유니폼을 입을지 선택할 수 있습니다.
MVP는 어떻게 확정되나요?
두 가지 경로가 있습니다. ① 운영진 1명 이상이 투표하면 즉시 확정 (투표율 무관), ② 일반 팀원이 실제 참석자의 70% 이상 투표하면 최다득표자 자동 확정. 70% 미달이면 운영진이 직접 지정할 수 있습니다. 자세한 내용은 7. 경기 기록의 MVP 섹션 참고.
AI 기능 사용 한도가 있나요?
기능별로 다릅니다. AI 풀 플랜은 경기당 2회 + 월 10회, AI 코치 분석은 경기당 3회, 자동 경기 후기는 경기당 1회 재생성, 회비 OCR은 무제한. 실패한 호출이나 룰 폴백은 한도에서 차감되지 않습니다. 자세한 내용은 11. AI 기능 참고.
무료인가요?
현재 모든 기능을 무료로 사용할 수 있습니다. 추후 운영 정책에 따라 유료 요금제가 도입될 수 있으며, 변경 시 사전에 안내드립니다. 먼저 함께한 팀에는 보호 정책이 적용됩니다.
