풋살/축구 팀 운영을 한 곳에서 — 총무를 위해 만들었습니다.

카카오로 시작하기 데모 체험하기

1가입하기

별도 회원가입 없이, 카카오 로그인 한 번이면 됩니다.

  1. pitch-master.app 접속
  2. 카카오 로그인 클릭
  3. 이름과 선호 포지션 입력 (연락처·생년월일은 나중에 해도 됩니다)
  4. 새 팀 만들기 또는 초대 링크로 기존 팀에 가입
가입 전에 먼저 둘러보고 싶다면 → 메인 페이지의 "30초 만에 데모 체험하기" 버튼을 눌러보세요. 가입 없이 모든 기능을 체험할 수 있습니다.

2팀 만들기 & 팀원 초대

  1. 팀명 입력 + 축구/풋살 선택 → 팀 생성
  2. 대시보드에서 "카카오톡으로 초대" 버튼을 눌러 팀원에게 공유
  3. 팀원이 링크를 눌러 카카오 로그인하면 자동으로 팀에 합류됩니다
📋 팀원에게 보낼 초대 안내 (복사해서 카톡에 붙여넣기)
우리 팀 참석투표/회비관리 앱이에요!
아래 링크 눌러서 카카오 로그인하면 바로 가입돼요 👇

👉 https://pitch-master.app/team?code=[초대코드]

※ 초대코드는 대시보드에서 확인할 수 있어요

팀원은 어떻게 가입하나요?

  1. 회장/총무가 보낸 초대 링크를 클릭
  2. 카카오 로그인 (3초)
  3. 이름과 선호 포지션 입력하면 자동 가입

별도 앱 설치 없이 브라우저에서 바로 사용할 수 있습니다.

초대 코드 없이 팀 찾기

초대 링크가 없어도 팀 이름을 검색해서 가입할 수 있습니다.

  1. 가입 후 팀 검색해서 가입을 선택
  2. 팀 이름을 검색하여 가입 신청
  3. 운영진이 승인하면 자동으로 팀에 합류됩니다

※ 회장이 설정 > 팀설정에서 "팀 검색 허용"을 켜야 검색에 노출됩니다.

3권한 체계

역할할 수 있는 것
회장모든 기능 + 팀 삭제 + 역할 변경 + 시즌 관리
운영진경기 등록/수정 + 회비 관리 + 멤버 관리 + 가입 승인
회원투표 + 기록 조회 + 게시판 + 개인 설정

회장이 멤버 관리에서 역할을 변경할 수 있습니다.

4참석 투표

매주 "이번 주 누가 와요?" 물어볼 필요 없이, 링크 하나로 참석을 자동 집계합니다.

경기 등록 → 팀원에게 알림 → 참석/불참/미정 투표 → 실시간 집계
· 경기를 등록하면 팀원에게 자동 알림이 갑니다
· 투표 마감 시간을 설정하면 마감 전날 리마인더도 자동 발송
· 운영진은 대리 투표도 가능 (카톡으로 참석 의사를 전달받은 경우)
· 미투표자를 한눈에 확인하고 투표 독려 알림도 보낼 수 있습니다

5회비 관리

엑셀이나 메모장으로 회비를 관리하던 시절은 끝. 통장 내역을 올리면 자동으로 정리됩니다.

통장 내역 올리기 → 이름·금액 자동 인식 → 납부 현황 자동 정리
· 엑셀 업로드: 은행 앱에서 다운로드한 거래내역 파일을 그대로 올리면 됩니다
· 스크린샷: 통장 거래내역 캡쳐 이미지를 올리면 글자를 자동 인식합니다
· 입금자 이름으로 팀원을 자동 매칭합니다
· 회원에게는 본인 납부 내역만 보여서 프라이버시가 보호됩니다
· 월별 납부 현황을 한눈에 확인할 수 있습니다
📂 엑셀 업로드 형식 안내
지원 은행: 카카오뱅크, 토스뱅크, 국민, 신한, 우리, 하나 등 대부분의 은행

은행 앱에서 내려받은 엑셀 파일을 편집 없이 그대로 올려주세요.
"거래일시"와 "거래금액" 항목을 자동으로 찾아서 처리합니다.

6스마트 라인업

경기장에 도착해서 포지션 정할 필요 없이, 버튼 하나로 자동 배치합니다.

참석 확정 → 자동 포지션 배치 → 쿼터별 균등 분배 → 카톡으로 공유
· 선호 포지션을 기반으로 자동 배치합니다
· 쿼터별로 균등하게 분배되어 모든 선수가 공정하게 뜁니다
· 전술판에서 드래그로 수정할 수 있습니다
· 완성된 라인업을 이미지로 저장하거나 카카오톡으로 공유할 수 있습니다

7경기 기록 & 통계

골, 어시스트, MVP를 경기 중에 바로 기록하고, 시즌 통계를 자동으로 집계합니다.

· 쿼터별 스코어보드로 득점/어시스트를 바로 기록
· 경기 후 팀원이 직접 뽑는 MVP 투표
· 시즌 골·어시·MVP·출석률 자동 집계
· 선수별 레이더 차트와 시즌 랭킹으로 한눈에 비교
· 경기 결과를 카카오톡으로 공유하거나 카드 이미지로 저장

8자체전 (A팀 vs B팀)

팀 내 홍백전을 할 때 사용합니다. 일반 경기와 동일한 기능에 팀 편성이 추가됩니다.

· 경기 등록 시 "자체전" 토글을 켜면 활성화
· 랜덤 배정 또는 수동으로 A/B팀을 나눌 수 있습니다
· A팀 vs B팀 별도 스코어보드와 팀별 골/어시스트 기록
· 전술판에서 A/B팀 탭 분리로 각 팀 포메이션 확인
· 기본적으로 시즌 통계에 포함되지 않지만, "통계 반영"을 체크하면 포함됩니다

9유니폼 안내

이번 경기에 홈/원정 유니폼 중 어떤 걸 입어야 하는지, 한눈에 알려줍니다.

· 설정 > 팀설정에서 홈/원정 유니폼 색상과 패턴을 지정
· 경기 상세 > 경기 정보에서 홈/원정 전환 버튼으로 이번 경기 유니폼 선택
· 대시보드와 경기 목록에서 유니폼 색상이 자동으로 표시됩니다
· "이번 주는 원정(빨강) 유니폼입니다" — 팀원이 헷갈릴 일이 없어요

11경기 운영 흐름 (한눈에 보기)

  1. 일정 등록 — 날짜, 시간, 장소, 쿼터 수, 유니폼(홈/원정) 설정
  2. 투표 받기 — 등록하면 팀원에게 자동 알림이 갑니다
  3. 라인업 짜기 — 투표 마감 후 "스마트 라인업" 버튼 한 번
  4. 경기 기록 — 골/어시스트/MVP를 실시간 기록
  5. 결과 공유 — 일지 탭에서 카카오톡이나 이미지로 공유
등록한 경기의 날짜·시간·장소·상대팀을 수정하려면 → 경기 상세 > 경기 정보 탭에서 "수정" 버튼을 누르면 됩니다. (운영진 이상)

12홈 화면에 추가하는 방법

브라우저 주소를 매번 입력할 필요 없이, 홈 화면에 추가하면 앱처럼 바로 접속할 수 있습니다.

아이폰 (Safari)

  1. Safari로 pitch-master.app 접속
  2. 하단 메뉴바에서 공유 아이콘 (네모에서 화살표 나온 모양 ⬆︎) 탭
  3. "홈 화면에 추가" 선택
  4. 우측 상단 "추가" 탭

안드로이드 (Chrome, 삼성 인터넷 등)

  1. 브라우저로 pitch-master.app 접속
  2. 하단에 뜨는 "홈 화면에 추가" 배너를 탭
  3. 또는 앱 안의 더보기 > 홈 화면에 추가 버튼 탭
홈 화면에 추가하면 알림(푸시)도 받을 수 있습니다. 설정 > 알림에서 켜주세요.

13자주 묻는 질문

Q. 앱 설치가 필요한가요?

아닙니다. 별도 앱 설치 없이 브라우저에서 바로 사용합니다. 홈 화면에 추가하면 일반 앱처럼 사용할 수 있고, 알림도 받을 수 있습니다. (위 12번 참고)

Q. 무료인가요?

현재 모든 기능을 무료로 사용할 수 있습니다. 다만 서버 운영 비용이 있어 향후 프리미엄 기능(고급 통계, 커스텀 브랜딩 등)은 유료로 전환될 수 있습니다. 지금 사용하시는 핵심 기능(투표, 회비, 라인업, 기록)은 계속 무료로 유지할 계획입니다.

Q. 여러 팀에 가입할 수 있나요?

네, 한 계정으로 여러 팀에 소속될 수 있습니다. 앱 안에서 팀을 전환하면 됩니다.

Q. 팀원이 앱을 안 깔아도 되나요?

네, 초대 링크를 클릭하면 브라우저에서 바로 카카오 로그인 → 가입됩니다. 별도 앱 설치가 필요 없습니다.

Q. 축구만 되나요? 풋살도 되나요?

축구와 풋살 모두 지원합니다. 팀 생성 시 종목을 선택하면 포메이션과 포지션이 종목에 맞게 설정됩니다.

Q. 데이터는 안전한가요?

모든 데이터는 한국(서울) 클라우드 서버에 암호화되어 저장됩니다. 통신도 모두 암호화(HTTPS)되어 있습니다.

Q. 카카오 인앱 브라우저에서 안 되는 게 있나요?

카카오톡 안에서 링크를 열면 일부 기능(알림, 홈 화면 추가)이 제한될 수 있습니다. 우측 상단 ··· → "외부 브라우저에서 열기"를 누르면 모든 기능을 사용할 수 있습니다.

기존에 사용하던 일정, 회비 내역, 멤버 목록 등 데이터 이관이 필요하시면 오픈채팅방에서 말씀해주세요. 엑셀이나 스프레드시트로 보내주시면 일괄 등록 도와드립니다.
