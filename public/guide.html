PitchMaster 시작 가이드

풋살/축구 팀 운영을 위한 올인원 관리 앱 — 총무를 위해 만들었습니다.

1가입하기 (1분)

  1. pitch-master.app 접속
  2. 카카오 로그인 클릭 (별도 회원가입 없음)
  3. 이름, 연락처, 선호 포지션 입력
  4. 새 팀 만들기 또는 초대 코드로 가입
가입 전에 먼저 체험해보고 싶다면 → 랜딩 페이지 하단의 "데모 체험" 버튼을 눌러보세요. 가입 없이 모든 기능을 둘러볼 수 있습니다.

2팀 만들기 (회장용)

  1. 팀명 입력 + 축구/풋살 선택 → 팀 생성
  2. 자동 생성된 초대 코드를 팀원에게 공유
  3. 팀원이 코드로 가입하면 자동으로 멤버에 추가됩니다
설정 > 팀설정에서 "팀 검색 허용"을 켜면, 팀원이 초대 코드 없이도 팀 이름을 검색해서 가입 신청할 수 있습니다. (운영진 승인 필요)

3권한 체계

역할할 수 있는 것
회장모든 기능 + 팀 삭제 + 역할 변경 + 시즌 관리
운영진경기 등록/수정 + 회비 관리 + 멤버 관리 + 가입 승인
회원투표 + 기록 조회 + 게시판 + 개인 설정

회장이 멤버 관리에서 역할을 변경할 수 있습니다.

4핵심 기능 3가지

⚡ 참석 투표
경기 일정 등록 → 팀원에게 자동 알림 → 참석/불참/미정 투표
실시간 집계되어 인원 파악이 즉시 됩니다. 대리 투표도 가능.
⚡ 회비 관리
통장 내역 엑셀 업로드 또는 스크린샷 OCR → 자동 매칭
월별 납부 현황이 한눈에. 회원에게는 본인 내역만 표시됩니다.
⚡ 스마트 라인업
참석 확정 인원 기반 → 선호 포지션 자동 매칭 → 쿼터별 균등 분배
전술판에서 드래그로 수정 가능. 이미지로 저장/공유도 됩니다.

5경기 운영 흐름

  1. 일정 등록 — 날짜, 시간, 장소, 쿼터 수 설정
  2. 투표 받기 — 등록 즉시 팀원에게 푸시 알림
  3. 라인업 짜기 — 투표 마감 후 스마트 라인업 생성
  4. 경기 기록 — 골/어시스트/MVP를 실시간 기록
  5. 결과 공유 — 일지 탭에서 카카오/이미지로 공유

7자주 묻는 질문

Q. 앱 설치가 필요한가요?

아닙니다. 웹앱이라 브라우저에서 바로 사용합니다. 홈 화면에 추가하면 앱처럼 사용할 수 있습니다. (PWA 지원)

Q. 무료인가요?

네, 모든 기능이 무료입니다.

Q. 여러 팀에 가입할 수 있나요?

네, 한 계정으로 여러 팀에 소속될 수 있습니다.

Q. 팀원이 앱을 안 깔아도 되나요?

네, 초대 링크를 클릭하면 웹에서 바로 카카오 로그인 → 가입됩니다.

Q. 데이터는 안전한가요?

Supabase(PostgreSQL) 클라우드 DB를 사용하며, 모든 통신은 HTTPS로 암호화됩니다.

기존에 사용하던 일정, 회비 내역, 멤버 목록 등 데이터 이관이 필요하시면 오픈채팅방에서 말씀해주세요. 엑셀/스프레드시트로 보내주시면 일괄 등록 도와드립니다.
