# PitchMaster 레퍼럴 이벤트 기획서

## "같이 차자! 챌린지"

> 기존 팀이 새로운 팀을 초대하고, 실제 활동까지 이어지면 양쪽 모두에게 보상하는 바이럴 마케팅 이벤트

---

## 1. 이벤트 개요

| 항목 | 내용 |
|------|------|
| 이벤트명 | 같이 차자! 챌린지 |
| 목적 | 신규 팀 유치 + 기존 팀 활성화 |
| 기간 | 2주 (시작일 미정) |
| 예산 | 10만원 |
| 대상 | PitchMaster에 등록된 모든 팀 (데모 팀 제외) |

---

## 2. 참여 흐름

```
기존 팀 회장 → 레퍼럴 코드 공유 (카톡/밴드/인스타 등)
                    ↓
새 팀 회장 → 가입 시 레퍼럴 코드 입력
                    ↓
새 팀이 조건 달성 (멤버 5명 + 경기 1회)
                    ↓
초대팀 + 신규팀 모두 보상 획득
```

### 조건 달성 기준
- 신규 팀에 **실멤버 5명 이상** 가입 (각각 다른 카카오 계정)
- **경기 1회 이상** 기록 (출석 체크 5명 이상인 경기)
- 가입 후 **2주 이내** 달성

| 예비 (배송비 등) | - | - | 2만원 |

## 4. 어뷰징 방지 규칙

### 문제점과 대응

| 어뷰징 시나리오 | 대응 방법 |
|----------------|-----------|
| 한 유저가 팀 여러 개 생성 | 이벤트 기간 중 1인 1팀 생성 제한 |
| 유령 팀 (혼자 가입) | 서로 다른 카카오 계정 5명 이상 필수 |
| 초대팀 멤버가 신규팀도 가입 | 초대팀-신규팀 간 멤버 중복 불허 |
| 사전등록만 채우기 | user_id 연결된 실멤버만 카운트 |
| 경기 기록 조작 | 출석 체크 5명 이상인 경기만 유효 |
| 카카오 부계정 돌려막기 | 전화번호 중복 체크 (선택사항) |

## 5. 홍보 계획

### 5-1. 직접 연락 (전환율 최고)

**대상**: 현재 등록된 5개 팀 회장/운영진

| 채널 | 방법 | 시기 |
|------|------|------|
| 카카오톡 1:1 | 각 팀 회장에게 이벤트 안내 + 레퍼럴 코드 직접 전달 | 이벤트 시작 1일 전 |
| 팀 공지 | 앱 내 공지게시판에 이벤트 안내 등록 | 이벤트 시작일 |

**메시지 예시**:
```
안녕하세요! PitchMaster입니다 ⚽

[같이 차자! 챌린지] 이벤트를 시작합니다!

주변에 풋살/축구 팀이 있다면 초대해주세요.
초대한 팀이 가입하고 경기 1회 기록하면
풋살공 + 기프티콘을 드립니다 🎁

👉 {팀이름}의 초대 코드: {코드}
👉 이벤트 기간: {시작일} ~ {종료일}

공유하기: {카카오 공유 링크}
```

### 5-2. 카카오톡 공유 템플릿

회장이 한 번 터치로 공유할 수 있는 **카카오톡 공유 메시지** 제공:

```
[PitchMaster - 팀 관리 앱]

우리 팀이 쓰고 있는 풋살 관리 앱인데,
지금 가입하면 풋살공 추첨 이벤트 중!

✅ 출석/회비/기록 자동 관리
✅ 경기 MVP 투표
✅ 팀원 생일 알림

초대 코드: {코드}
가입하기: https://pitch-master.app?invite={코드}
```

### 5-3. 온라인 커뮤니티 (비용 0)

| 채널 | 방법 | 예상 도달 |
|------|------|-----------|
| **네이버 카페 "풋살인"** | 이벤트 홍보글 + 앱 소개 | 회원 수만명 |
| **네이버 카페 "풋살매치"** | 이벤트 홍보글 | 회원 수만명 |

| **인스타그램** | 이벤트 카드뉴스 1장 + 스토리 | 해시태그 노출 |
| **에브리타임 (대학)** | 대학 풋살 동아리 대상 | 학교별 |

**커뮤니티 홍보글 구성**:
```

출석, 회비, 경기 기록을 한 곳에서 관리하는
PitchMaster라는 앱을 소개합니다.

지금 가입하면 이벤트 참여 가능합니다.

📌 주요 기능
- 경기 출석/불참 투표
- 회비 납부 관리
- 경기 기록 + MVP 투표
- 팀원 생일 알림
- 카카오 로그인 (가입 10초)

📌 이벤트
- 기간: {시작일} ~ {종료일}
- 팀 가입 + 경기 1회 기록 시 풋살공 추첨
- 가입: https://pitch-master.app
```

## 7. 성과 측정 지표 (KPI)

| 지표 | 목표 |
|------|------|
| 신규 팀 가입 수 | 5팀 이상 |
| 조건 달성 팀 수 | 3팀 이상 |
| 신규 가입 유저 수 | 25명 이상 (5팀 x 5명) |
| 이벤트 후 잔존율 (4주 뒤) | 신규 팀 50% 이상 활동 유지 |
| 레퍼럴 코드 공유 횟수 | 추적 가능 시 측정 |

## 8. 이벤트 종료 후 후속 조치

1. **성과 리뷰**: 목표 달성 여부 + 어뷰징 발생 여부 점검
2. **잔존율 관리**: 신규 팀에게 2주 뒤 "잘 쓰고 계신가요?" 알림
3. **상시 레퍼럴**: 이벤트 종료 후에도 뱃지 보상은 유지 → 상시 초대 동기 부여
4. **다음 이벤트 검토**: 성과에 따라 2차 이벤트 예산/규모 결정
