## 3. 경쟁 앱 관찰 노트

### 3-1. 모이라니 (moirani) — 회비 UX 인상적
**2026-04-21 스크린샷 관찰**:
- 홈 화면: 팀 카드 리스트 (축구·팀관리자 라벨), 우측 상단 "구독 상태: 미구독" 명시 → **Free + 구독 전환 모델**
- 회원별 납입 현황: **12개월 가로 매트릭스**, 7단계 상태 라벨(납입/미납/예정/면제/가입 전/부상/휴면)
- 월별 감사 보고서: 별도 사이드 카드로 월 누적 수입·지출·소계·**마감 잔액** 한눈에
- 수입/지출 거래 상세: 날짜·구분·메모·금액 4열 표 (필터 최소)

**우리 대비 강점**:
1. **부상/휴면/면제 상태** — 일시적 사정을 납부 면제로 기록. 우리는 이게 없어서 "미납"과 "면제"를 구분 못 함
2. **월 마감 잔액 보고서** — 총무가 월마다 "이번 달 얼마 남았지?" 확인하는 흐름이 한 화면. 우리는 월 요약 UI 없음

**우리 대비 약점**:
- OCR 없음 (추정) — 입금 대조는 수기 추정
- 전술·라인업·AI 전무
- PC 미지원 (네이티브)

### 3-2. 축구고 — 선수 개인 성장 중심
`docs/production-readiness.md:69-85` 참조. GPS·Wear OS·OVR 특화, 총무 운영 기능 약함. 우리 포지셔닝 "선수 개인은 축구고, 팀 운영은 PitchMaster" 는 이 분석에서 나옴.

### 3-3. 조기사커 (JogiSoccer) — jogisoccer.com
**2026-04-21 앱스토어 + 웹 심층 조사**:

**출시 현황**
- App Store ID: `id6760828190` — 2026-04-01 출시 (약 3주 전), iOS 전용
- 현재 버전: **v1.1.0** (2026-04-20 업데이트). 리뷰 **0건** — 실질 사용자 극소수 단계
- Android 미출시. 웹 대시보드 별도 제공

**버전 업데이트 이력 (빠른 이터레이션)**
| 버전 | 주요 추가 기능 |
|------|---------------|
| v1.0.7 이전 | 기본 팀 관리, 참석 투표, 실시간 경기 기록 |
| v1.0.8 | **MVP 순위 시스템** 추가 |
| v1.0.9 | **3D 경기장 시뮬레이터**, 유니폼 커스터마이징 |
| v1.1.0 | **동료 평가 프리셋**, 라인업 공유 기능 |

→ 약 3주 만에 4회 업데이트. 개발 속도 빠름.

**핵심 차별화 기능**
1. **헝가리안 알고리즘 자동 포지션 배치** — 선수별 선호 포지션 데이터 기반, 팀 전체 최적 매핑. 우리 AI Full Plan과 직접 경쟁.
2. **58개 항목 익명 동료 능력치 평가 + 레이더 차트** — 가중 알고리즘 적용. 평가 프리셋(v1.1.0) 추가로 UX 개선 중. "서열 갈등 없는 객관적 실력 파악" 컨셉.
3. **이식 가능한 선수 커리어** — 팀 이적 후에도 커리어 데이터 선수 본인 소유. 플랫폼 락인 전략 (선수 개인 유입 채널).
4. **자체 리그 운영 + 팀 밸런싱** — 아마추어 리그 사무국용. 우리에게 없는 B2B 진입로.
5. **용병·심판 구인 게시판** — 플랫폼 내 인력 매칭. 네트워크 효과 강화.
6. **3D 경기장 시뮬레이터** (v1.0.9) — 시각적 차별화 포인트. 실용성보다 와우 효과.
7. **다국어 5개** (한/영/일/중/스페인어) — 해외 확장 준비. 우리는 한국어 전용.
8. **Excel 업로드/다운로드** — 회원 일괄 등록·내보내기. 우리가 없는 기능.
9. **회비 QR 결제** — 앱 내 결제 연동. 우리는 OCR 파싱 방식.
10. **macOS + visionOS 네이티브 지원** — 애플 생태계 전방위.

**가격 — 주의: 랜딩페이지와 앱스토어 문구 상충**
- 웹 랜딩: "완전 무료" 명시
- 앱스토어 설명: "15명 이하 무료" (정식 출시 후 유료 전환 시사)
- 현재 베타 기간 무료 운영 중. 정식 출시 시 소규모 팀만 무료일 가능성.
- → 영구 무료 단정 금지, 지속 모니터링 필요

**우리 대비 강점**:
1. **헝가리안 알고리즘 자동편성** — 수학적 최적화. 우리 AI Full Plan과 직접 충돌
2. **익명 동료 평가 시스템** — 리텐션·게임화 핵심. 우리에게 없음
3. **선수 소유 커리어** — 선수 개인 유입 + 이탈 방지 구조
4. **Excel I/O** — 회원 대량 등록 및 데이터 이동성
5. **리그 운영** — 우리가 포지셔닝하지 않은 B2B 리그 시장
6. **다국어·macOS** — 글로벌/멀티플랫폼 커버리지

**우리 대비 약점**:
- **AI 전술 코치·경기 후기** 없음 (개발 중 언급) — 현재 PitchMaster가 선점
- **OCR 회비 파싱** 없음 — 입금 대조는 QR 결제 or 수기
- **Android 미출시** — 국내 안드로이드 점유율(약 72%)에서 완전 공백
- **리뷰 0건** — 실검증 사용자 없음. 안정성·UX 미검증
- **PWA 없음** — 설치 필수 (iOS App Store)
- **베타 무료** — 정식 출시 후 가격 정책 불투명

**위협 수준: 🟡 모니터링** (현재는 낮음, 6개월 내 위험 고조 가능)
- 현재: 리뷰 0건, Android 없음, 베타 단계 → 실 사용자 풀 미형성
- 6개월 내: Android 출시 + MVP·리그 기능 안정화 시 직접 경쟁 전선 확대
- 주시 신호: Android 출시 발표 / 리뷰 10건 돌파 / AI 기능 출시 공지
