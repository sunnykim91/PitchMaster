# 경쟁 앱 분석 — 조기축구·풋살 팀 관리 / 팀 매칭 카테고리

> **목적**: 국내 경쟁 앱을 주기적으로 추적하고, 기능 gap 을 우리 개선 백로그에 연결한다.
> **갱신 주기**: 월 1회 (기능·가격·다운로드 수 변화 확인)
> **최종 갱신**: 2026-07-01 (웹 전수 재조사 — 앱 15+종 App Store/Play/공식사이트 직접 확인)
> **직전 갱신**: 2026-04-21

> ⚠️ **경쟁 앱 지표 신뢰도 주의**: App Store/Play는 정확한 설치 수를 숨기고, 웹사이트 통계 카운터는 JS 렌더링이라 직접 fetch로 확인이 안 되는 경우가 많다. 아래 수치는 대부분 **자사 발표·2차 집계·스토어 리뷰 수** 기반이며 "확인됨/추정/확인불가"를 구분해 표기한다. 인용 전 재확인.

## 0. 2026-07-01 재조사 핵심 변화 (TL;DR)

지난 2.5개월간 시장이 크게 움직였다. 4월 대비 달라진 것:

1. 🔴 **조기싸커 — Android 정식 출시 + AI 진입.** 5월엔 "iOS 전용·AI 전무"였으나, Google Play 출시(`com.jogisoccer.app`, 6/30 업데이트)·AI 코치 노트(6/7)·전술보드(애니메이션·MP4)·슛맵/히트맵을 2개월 만에 다 붙였다. **우리가 5월에 정한 1순위 감시 체크포인트(Android 출시)가 실제 발동**. 위협도 상승.
2. 🔴 **축구고 — 회비 장부 추가 + AI 하이라이트 예고.** 우리 메모리의 "축구고=회비 없음·선수개인 성장만" 포지셔닝이 **양 축 다 깨졌다**. 6월 말 v1.3.8에 회비 장부(수입·지출·잔고) 탑재, AI 하이라이트 자동편집 베타 예고. H1에만 앱스토어 25개 버전(6월에만 8개)으로 개발 속도 공격적.
3. 🟡 **신규 직접 경쟁 팀관리 앱 3종 발굴** — **FootballLab(풋볼랩)**, **축구생활(SoccerLife)**, **에이풋볼(A-FOOTBALL)**. 1인/소규모 개발자 앱이 계속 진입해 팀관리 세그먼트가 붐빔.
4. 🟢 **매칭 앱 → 팀관리 융합 확정.** 플랩풋볼이 "팀 서비스·팀 리그" 출시, 아이엠그라운드가 "아이엠매치(팀매치)" 출시(2025). 다만 회비·출석·기록의 **깊은 팀운영 툴킷은 아직 없음** = 방향성 위협.
5. ⚪ **오분류·확인불가 정정.** 플레이어스·위볼 = 축구 앱으로 실존 확인 불가(팀웨어 쇼핑몰/미트볼집 등 동명). 그라운드N = 넥슨 FC온라인 후원 캠페인(앱 아님). 팀러 = 영국 Pitchero 글로벌 앱(한국 아님). 모이라니 = 실존하나 Android 전용·풀 팀운영앱(회비 중심 아님).
6. 🟡 **사말라(Samala) 신규 발굴 — 우리와 포지셔닝 최근접.** 남양주 조기축구팀 **고인돌FC**가 총무 pain(참석·회비·기록 독촉)을 풀려 직접 만든 **웹 기반** 팀운영 앱. 카카오 로그인·자체제작·자사 ~100팀(추정)까지 **우리 기원 서사와 거의 동일**. 차별 기능 = **도착 셀프 체크인→도착순 스쿼드 자동**·**미납자 투표 하드 제한**·경기장 카카오네비. 단 AI·회비 OCR·벌금/면제 깊이·후기 생성은 없음(카피 기준) = 우리 해자 유효.

## 5. 포지셔닝 정리 (2026-07 업데이트)

**우리 단독 강점 (경쟁 앱 아직 못 따라온 것):**

3. **PWA·PC 접근성** — 네이티브 앱(축구고·동네축구·FM조축·모이라니)들 구조적으로 못 따라옴 (단 **사말라·FootballLab·조기싸커 웹 대시보드**는 웹 peer — 접근성 자체는 동급이라 이제 단독 강점 아님, 우리 차별은 AI·회비 깊이로 이동)
4. **자체전(A vs B)** + **운영진 전용 정체성**(no-ranking 친목 보호)

**경쟁 앱이 우리보다 앞선 것 (검토):**
1. **다국어·글로벌** (조기싸커 5개·축구고 3개) — 1인 운영 한계로 우리 보류

3. **매칭·용병 마켓플레이스·리그** (조기싸커·플랩·축구고) — 우리 미보유. 100팀+ 정착 후 검토
4. **소셜 피드/스토리** (축구생활) — AI 후기와 결합 가능성
