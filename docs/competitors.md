> **목적**: 국내 경쟁 앱을 주기적으로 추적하고, 기능 gap 을 우리 개선 백로그에 연결한다.
> **갱신 주기**: 월 1회 (기능·가격·다운로드 수 변화 확인)
> **최종 갱신**: 2026-07-12 (2차 재조사 — 조기싸커 유료광고 확정 + 3일 델타 + 신규진입 스캔)

> ⚠️ **경쟁 앱 지표 신뢰도 주의**: App Store/Play는 정확한 설치 수를 숨기고, 웹사이트 통계 카운터는 JS 렌더링이라 직접 fetch로 확인이 안 되는 경우가 많다. 아래 수치는 대부분 **자사 발표·2차 집계·스토어 리뷰 수** 기반이며 "확인됨/추정/확인불가"를 구분해 표기한다. 인용 전 재확인.

### 6-2. 경쟁사 마케팅 채널·크리에이티브
- **두 카테고리가 다르게 마케팅**: 매칭앱(플랩·매치업)=B2C "즉시 재미/최저가"·인스타 UGC 바이럴·VC 자본 / 팀관리앱(축구고·동네축구·조기싸커)=오가닉·커뮤니티 주도·유료광고 미미
- **플랩** = 인스타 태그공유 이벤트로 스케일(유료광고 최소). 지역 리그+오프라인 밋업으로 애착
- **매치업** = "풋살구장 우주 최저가" 가격 소구
- **축구고** = 기능/유틸 소구 + **GPS/히트맵 시그니처 훅** + "조기축구 용병 매칭" SEO 랜딩
- **조기싸커** = 다음 카페 개발자 직접 회고글(1인 커뮤니티 침투)
- **고알레(GoAle)** = 🌟 **콘텐츠 퍼스트 GTM** — 조기축구 드론 촬영 유튜브 콘텐츠 → 브랜드/굿즈 → 앱. 퍼포먼스 광고 대신 콘텐츠 주도 성장. 참고 템플릿 가치 큼
