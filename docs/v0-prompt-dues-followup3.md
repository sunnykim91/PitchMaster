# v0 추가 프롬프트 3차 — 최종 마무리

거의 완성입니다. 아래 5가지만 수정/추가해주세요.

---

## 1. 납부현황 — 미납 금액 색상 구분

현재 미납 회원의 금액(20,000원)과 납부 완료 회원의 금액(30,000원)이 같은 텍스트 색상입니다. 구분이 안 됩니다.

수정:
- 미납 회원 금액: `color: hsl(0, 65%, 60%)` (loss 색상, 빨강)
- 납부 완료 회원 금액: `text-muted-foreground` (회색, 강조 불필요)
- 면제 회원: 금액 표시 없음

---

## 2. 프로그레스 바 — 단색으로 확실하게

납부 통계 프로그레스 바에 아직 미세한 그라데이션이 남아있습니다.

수정:
- fill 색상: `background-color: hsl(152, 55%, 55%)` 단색 (gradient 속성 완전 제거)
- 배경: `background-color: hsl(0, 0%, 15%)`
- `background-image: none` 명시

---

## 3. 평회원 뷰 — 별도 섹션으로 보여주기

현재 운영진 뷰만 보입니다. 평회원이 접속했을 때의 화면을 별도 섹션으로 추가해주세요:

차이점:
- 탭 바: **"입출금" 탭 1개만** 표시 (납부현황, 내역 올리기, 설정 숨김)
- 잔고 카드와 탭 바 사이에 **"내 납부 상태"** 카드 추가
- 나머지(잔고 카드, 입출금 내역)는 동일

내 납부 상태 카드:
```
┌─────────────────────────────────────┐
│ 내 납부 상태              < 04 >    │
│                                     │
│  ✅ 납부 완료              30,000원 │
│     (success 색상)                  │
└─────────────────────────────────────┘
```

또는 미납일 때:
```
┌─────────────────────────────────────┐
│ 내 납부 상태              < 04 >    │
│                                     │
│  미납                     30,000원  │
│  (loss 색상)       [입금했어요 💬]  │
│                    (outline 버튼)   │
└─────────────────────────────────────┘
```

- 카드 스타일: rounded-xl, bg-card, border border-white/4
- 월 이동 `< >`: min-h-[44px] 터치 타겟
- "입금했어요" 버튼: outline variant, border loss 색상

---

## 4. 빈 상태 — 입출금 탭 예시

입출금 탭에 거래 내역이 없을 때의 빈 상태를 보여주세요. 거래 카드 리스트 자리에 표시됩니다:

```
        📋 (48px 아이콘, muted)

   아직 거래 내역이 없습니다
      (text-base, font-semibold)

 내역 올리기 탭에서 스크린샷이나
 엑셀을 올려 거래 내역을 추가하세요
   (text-sm, text-muted-foreground)

      [ 내역 올리기 ] (primary 버튼)
```

- 전체 영역: `py-16 text-center`
- 아이콘: `h-12 w-12 text-muted-foreground/40 mx-auto`
- CTA 버튼: primary variant, 클릭 시 "내역 올리기" 탭으로 이동

---

## 5. 수기 입력 — 접힌 상태 디자인

현재 항상 펼쳐져 있는데, 기본은 접힌 상태여야 합니다. 접힌 상태와 펼친 상태를 모두 보여주세요:

접힌 상태:
```
┌──────────────────────────────────┐
│ 수기 입력                    ▼   │
└──────────────────────────────────┘
```

- 한 줄 바: `rounded-xl bg-card border border-white/4 px-4 py-3`
- 우측 화살표: ChevronDown 아이콘, 펼치면 ChevronUp으로 변경
- 클릭하면 아래로 펼쳐지며 현재 폼 UI 표시 (animate-in slide-in-from-top-2)

펼친 상태는 현재 디자인 그대로 유지하되, 상단에 "수기 입력 ▲" 헤더 추가
