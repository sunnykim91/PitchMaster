# 0명에서 180명으로 — 조기축구 팀 관리 서비스 사용자를 어떻게 모았나

> velog 시리즈 3편 — 풋살/조기축구 총무가 직접 만든 팀 관리 서비스 PitchMaster 성장기

---

## 시작점: 우리 팀 하나

1편과 2편에서 썼던 것처럼, 처음엔 저희 팀 하나였어요. 매주 카톡 공지방 참석 투표·엑셀 회비 정리가 지긋지긋해서 직접 만든 웹앱. 기능이 어느 정도 돌아가자 다른 팀도 쓸 수 있게 열어보기로 했습니다.

문제는 "어떻게 알릴까" 였어요. 저는 개발자고, 마케터가 아닙니다. 광고 돌릴 돈도 없고요.

---

## 첫 외부 팀이 들어온 순간

처음 외부 팀이 가입한 건 지인 루트였어요. "너희 팀 총무 힘들어하던데 이거 써봐" 한 마디. 그 팀이 한 주 써보고 "괜찮네" 했을 때, **시스템이 한 명만의 도구가 아니라는 걸 확인**한 순간이었습니다.

그때까지 머릿속에 막연히 있던 가설이 있었어요. "조기축구 총무는 다 비슷한 일로 고생한다." 우리 팀과 완전히 다른 팀도 똑같이 "회비 정리 너무 귀찮다" 고 말하는 걸 듣고 확신했죠.

---

## 네이버 카페가 생각보다 컸다

본격적으로 외부 유입을 시작한 건 네이버 조기축구 카페였어요. 몇 개 카페에 **경험담 형식의 글**을 올렸습니다.

광고 티 내면 안 됩니다. 조기축구 카페는 스팸에 민감해요. 그래서 글은 이런 형식으로 갔습니다:

- 제목: "총무 4년차입니다. 참석확인 어떻게 하세요?"
- 본문: 우리 팀에서 매주 겪던 문제 공감 → 이것저것 시도했는데 안 됨 → 그래서 만들어 써봄 → 팀원 반응 → 본문 마지막에 한 줄 링크

반응이 극과 극이었어요. 어떤 카페는 댓글이 50개 넘게 달렸고, 어떤 카페는 광고로 신고당해 글 삭제됐습니다. 신고당한 카페엔 다시 안 올렸어요. 반응 좋은 카페만 남겨두고 집중.

---

## 총무한테 먹히는 메시지

여러 카피를 돌려본 끝에 **"30분 → 30초"** 가 가장 잘 꽂혔어요.

- 참석 확인: 30분 → 30초
- 회비 정산: 30분 → 1분
- 라인업: 20분 → 3초

시간 절감이 가장 구체적이고 체감 가능했거든요. "효율적이에요" 보다 "30분이 1분 된다" 가 훨씬 먹힙니다.

이걸 랜딩 Hero 섹션에 배지로 박제했고, 카페 글에도 이 숫자 3개를 앞세웠어요. **추상적 장점은 스킵되지만 구체 숫자는 기억에 박힙니다.**

---

## 데모 모드가 전환율을 두 배로 올렸다

초기엔 카카오 로그인 → 팀 생성 → 메뉴 확인 순서로 가입했어요. 전환율이 낮았죠. 로그인 단계에서 이탈하는 사람이 많았습니다.

해결: **"데모 시작" 버튼**. 로그인 없이도 샘플 팀 데이터로 바로 앱을 체험할 수 있게 했어요. 데모엔 FC DEMO 라는 가상 팀이 있고, 경기 기록·회비 이력·AI 기능까지 전부 구동됩니다.

결과:
- 데모 도입 전: 랜딩 방문 100 → 로그인 시도 12 → 팀 생성 7
- 데모 도입 후: 랜딩 방문 100 → **데모 시작 34** → 로그인 시도 18 → 팀 생성 11

데모를 본 사람이 로그인까지 가는 비율이 높아요. "일단 뭐가 되는지 보자" 의 힘입니다.

## 소셜프루프: 팀 수 카운터 하나의 효과

랜딩 Hero 바로 밑에 `N개 팀 · M+ 회원이 사용 중` 배지를 넣었어요. 처음엔 이거 없이 돌렸는데, 배지 추가 후 가입률이 눈에 띄게 올라갔습니다.

이유는 단순해요. 조기축구 총무는 **실패 위험을 극도로 경계**합니다. "도입했다가 팀원들이 귀찮아하면 어쩌지" 가 제일 큰 저항선이에요. 다른 팀이 이미 쓰고 있다는 신호 하나가 그 저항을 확 낮춥니다.

82개 팀이 쓰고 있으면 "우리도 써도 되겠네" 가 자연스럽게 나오거든요.

---

## 입소문 루프

팀 총무 1명이 가입하면 그 팀 회원 20~30명이 자동으로 링크를 받고 들어옵니다. 링크 눌러 본인 참석만 클릭하는 것도 진입 장벽이 낮고요.

그중에 "우리 팀도 이거 쓸래" 하는 다른 팀 총무가 나옵니다. 처음엔 전혀 예상 못 한 흐름이었는데, 같은 카페 안에서 추천이 돌면 가입 수가 톡 튀어요.

**카카오톡 공유 기능**이 특히 컸어요. 참석 투표 링크를 카톡방에 공유하면 그 방의 다른 사람들이 PitchMaster 를 처음 보게 됩니다. 의도 없는 광고 채널이 하나 생긴 셈.

---

## 실패한 시도들

1. **인스타 광고** — 예산 10만원 돌렸는데 가입 1명. 조기축구 총무가 인스타에서 팀 관리 앱을 찾지는 않더군요.
2. **유튜브 간접 홍보** — 조기축구 채널 영상에 댓글 달기. 스팸 신고만 여러 번. 폐기.
3. **페이스북 그룹** — 반응 거의 없음. 연령대가 맞지 않았어요.

**돌아보면 채널 하나만 집중하는 게 답**이었어요. 네이버 조기축구 카페 + 입소문 루프, 이 두 가지에 집중한 뒤부터 성장이 꾸준해졌습니다.
