## 공통 결론 (5사 교차)
- **회비 깊이 = 우리 단독 재확정**: 조기싸커·축구고 = 기본 회비(납부/미납/면제 현황 + 수입지출 장부)만. **OCR·벌금·선납 전부 없음**(App Store 본문 verbatim). 스쿼드온·플랩 = 회비 아예 없음.
- **AI 실배포 = 우리 앞섬**: 축구고 AI 하이라이트 "다음 업데이트 예정"(미출시), 조기싸커 AI 코치노트는 있으나 헤드라인 카피에서 탈락(안 밀음).
- **경쟁사 오가닉 콘텐츠 발자국 ≈ 0**: 조기싸커·축구고 둘 다 유튜브·네이버블로그·카페 실사용 후기 0건. → 우리 SEO/블로그 자산이 진짜 차별화(유기 채널 우선 전략 강화 근거).

## 1. 조기싸커 (JoGi Soccer) — 델타 큼
- **App Store ID `id6760828190`** (판매자 Youngbin Ha), 첫 출시 2026-04-01, 한/미 단일 리스팅. **최신 v1.5.2 = 2026-07-10**. 무료·IAP 0.

- **회비 verbatim**: "회비 관리 — 납부·미납·면제 실시간 현황, 수입·지출 카테고리별 투명 공개." → **OCR·벌금·선납 없음 확정.**
