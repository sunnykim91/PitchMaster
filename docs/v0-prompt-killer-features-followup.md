## 🔧 보정 1 — `PlayerCard` 데모 페이지 레이아웃 재구성

### 지금의 문제
4종 등급 카드(ICON/HERO/RARE/COMMON)를 **한 줄 4열 그리드**로 배치해서, 카드 한 장당 폭이 너무 좁습니다. 그래서:
- 카드 안에 OVR 숫자와 포지션 라벨만 보이고, 이름·시그니처·스탯이 모두 잘려서 안 보임
- RARE 카드는 이름이 두 줄로 줄바꿈됨
- COMMON 카드는 우측이 잘림
- EA FC 카드의 핵심인 "한 장의 풍성함"이 사라짐

### 새 레이아웃 요구
**4장을 동시에 보여주되, 한 장당 최소 280×380px 이상이 되도록** 재배치:

- **모바일(375px~)**: 세로 stack — 한 줄에 1장씩, 카드 폭 320px 정도, 위에서 아래로 ICON → HERO → RARE → COMMON 순
- **태블릿(640px~)**: 2×2 그리드 — 한 장당 280×380px
- **데스크탑(1024px~)**: 2×2 그리드 또는 4열이지만 카드 폭 280px 이상 보장 (전체 폭 1280px+)

또는 더 추천하는 변형:
- **ICON 1장을 hero로 거대하게**(450×600) 페이지 상단에 배치하고
- **그 아래 HERO/RARE/COMMON 3장은 작은 thumbnail(220×290)** 비교용으로 한 줄에 나열

→ 어느 쪽이든 좋으니, **카드가 작아서 안의 내용이 잘리는 일이 없게** 해주세요.

## 🔧 보정 2 — `PlayerCard` 컴포넌트 자체에 빠진 요소 추가

지금 카드 안에는 OVR 숫자, 포지션 라벨, rarity 라벨, 작은 이름 정도만 들어가 있습니다.
원래 명세에 있던 아래 4가지가 빠져 있으니 추가해주세요.

### A. 거대한 등번호 워터마크
- 카드 중앙 배경에 **등번호를 거대하게(폰트 사이즈 280~340px) 반투명(opacity 0.12~0.18)** 으로 배치
- 예: `10`, `7`, `9` — 카드의 시각적 중심
- 이 위에 선수 이름이 얹어짐 (z-index로 처리)
- `photoUrl` prop이 있으면 이 워터마크 대신 사진 표시 (지금은 photoUrl 없는 mock 기준)

### B. 시그니처 한 줄 캐치프레이즈
- 선수 이름 바로 아래에 **한 줄 카피**:
  - ICON 예: `"15골 8어시 — 시즌 MVP 후보"`
  - HERO 예: `"5경기 연속 MOM에 빛난 미드필더"`
  - RARE 예: `"22경기 출장 · 클린시트 7회"`
  - COMMON 예: `"꾸준함으로 시즌을 채운 선수"`
- 폰트 사이즈 13~14px, opacity 70%, italic 가능
- props에 이미 `signature?: string` 필드 있음. mock 데이터 4종 모두 채워주세요.

### C. Hero stat 1개 강조 + 컨텍스트 chip
- 6개 스탯을 균등 그리드(3×2)에 박지 말고, **포지션의 시그니처 스탯 1개를 큰 카드로 강조**:
  - FW → 골 (예: `15` 48px+)
  - MID → 어시
  - DEF → 클린시트
  - GK → 클린시트
- 이 hero stat 옆에 **컨텍스트 chip**: `🏆 팀 1위`, `📈 시즌 최다`, `🔥 13경기 연속`, `상위 5%`
- 나머지 5개 스탯은 그 아래 작은 그리드(예: 5열 또는 2×3)로 작게
- props `stats` 배열의 `isHero: true` 플래그를 가진 항목을 hero로 렌더

mock 데이터 예시 (ICON 카드):
```ts
{
  ovr: 92,
  rarity: "ICON",
  positionLabel: "FW",
  positionCategory: "FW",
  playerName: "김민수",
  jerseyNumber: 10,
  teamName: "FC 피치마스터",
  teamPrimaryColor: "#e8613a",
  seasonName: "2026 시즌",
  signature: "15골 8어시 — 시즌 MVP 후보",
  stats: [
    { label: "골",     value: "15",  rank: "🏆 팀 1위",      isHero: true },
    { label: "어시",   value: "8",   rank: "상위 10%" },
    { label: "공격P",  value: "23",  streak: "🔥 5경기 연속" },
    { label: "MOM",    value: "5" },
    { label: "출석률", value: "82%", streak: "13경기 연속" },
    { label: "경기",   value: "18" },
  ],
}
```

### D. Rarity별 시각 차이 더 강하게
지금은 4종 카드의 색만 다르고 형태가 똑같습니다. **ICON과 COMMON은 시각적으로 명확히 다른 무게감**이 필요해요:

- **ICON (92)**:
  - 배경에 골드 conic-gradient + 미세 노이즈
  - 두꺼운 골드 메탈릭 double border (안쪽 어두움 + 바깥 빛남)
  - 카드 외곽에 골드 글로우 `box-shadow: 0 0 80px rgba(255, 215, 0, 0.4)`
  - 카드 면에 sparkle dot 5~10개
  - 상단에 `★ ICON ★` 또는 `LEGENDARY` 라벨
- **HERO (84)**: 코랄 메탈릭 테두리 + 코랄 글로우, sparkle 없음
- **RARE (76)**: 청록 holographic 그라디언트 + teal 글로우
- **COMMON (62)**: 무광 단색, 글로우 없음, 일부러 평범하게 (대비용)

ICON 카드 옆에 COMMON 카드를 두면 **"이 둘은 같은 카드 시스템이지만 격이 다르다"** 가 한눈에 보여야 합니다.

## 🔧 보정 3 — `PlayerProfilePage` 히어로 섹션을 진짜 영화 인트로로

### 지금의 문제
- 히어로 섹션이 압축되어 있어서 "영화 인트로" 임팩트가 약함
- 이름 폰트가 충분히 거대하지 않음
- 배경이 단조로움 (스타디움 라이트, 축구장 라인 패턴 안 보임)
- 그냥 평범하게 "이름 + 부제 + 카드"가 쌓여 있음

### 새 요구사항

**A. 진짜 100vh 점유**
- 히어로 섹션이 화면 첫 1뷰 거의 전체를 차지하게 (`min-h: 100vh` 또는 `90vh`)
- 스크롤 전에는 히어로만 보이고, 시즌 다이제스트는 스크롤해야 등장

**B. 거대 타이포그래피**
- 선수 이름 폰트 사이즈: `clamp(56px, 14vw, 120px)`
- 한글 자간(`letter-spacing`) 넓게 (0.05em~0.1em)
- font-weight 900, 거의 디스플레이 폰트 느낌
- 텍스트 그림자나 미세한 글로우로 입체감

**C. 배경 dramatic하게**
- 팀 컬러(`teamPrimaryColor`) 그라디언트 (좌상단 밝음 → 우하단 어두움)
- **흐릿한 스타디움 라이트** — 화면 위쪽에서 아래로 향하는 radial light 2~3개 (하얀색 30% opacity, blur 60px+)
- **축구장 라인 패턴** — SVG로 페널티박스 라인을 그려서 5% opacity로 배경에 깔기 (또는 단순 grid line)
- 노이즈 텍스처 미세하게
- vignette — 외곽이 어두워져서 시선이 중앙으로

**D. 위계 정리**
- 상단 작은 overline: `2026 시즌` (12px, opacity 50%, letter-spacing 넓게)
- 중앙 거대 이름 (위 B)
- 그 아래 한 줄 부제: `FC 피치마스터 · FW · #10 · 18경기 출장` (16px, opacity 70%)
- 역할 뱃지: 주장/부주장 (있을 때만, 작은 코랄 chip)
- 그 아래 **시그니처 한 문장** (자동 생성, 코랄 색):
  - `"15골 8어시 — 팀 득점왕"`
  - `"승률 82%, 그가 뛰면 팀은 이긴다"`
  - 폰트 사이즈 18~20px, italic, primary color
- 우상단: 작은 `PITCHMASTER` 워터마크 (10px, opacity 30%, letter-spacing 0.3em)
- 하단 중앙: 아래로 향한 화살표 + `SCROLL` 텍스트 (scroll cue, 살짝 bouncing 느낌)

**E. 영화 포스터 같은 분위기**
- "잘 만든 페이지 헤더"가 아니라 **"이 사람이 시즌의 주인공이다"** 라고 외치는 한 화면
- 보는 사람이 스크린샷 찍어서 카톡 프로필에 박고 싶어야 함

## ⚠️ 가장 중요한 한 가지

**카드 안이 비어있는 게 가장 큰 문제였습니다.** 카드를 작게 만들 거면 안의 요소를 줄이고, 안의 요소를 다 넣을 거면 카드를 키워주세요. **OVR 숫자만 떠있는 카드는 EA FC 카드가 아닙니다.**

이름·등번호 워터마크·시그니처 캐치프레이즈·hero 스탯·컨텍스트 chip — 이 5가지가 한 카드 안에 모두 있어야 "갖고 싶은 한 장"이 됩니다.
