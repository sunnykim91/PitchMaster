# 조기축구·풋살 유튜브 아웃리치 실행안 (2026-07-01)

> 목적: 유료 인스타 스케일 대신, **고의도 트래픽 + 우리 카테고리에서 검증된 콘텐츠·인플루언서 채널**로 유입 확보.
> 근거: `competitors.md` 6장(고알레 콘텐츠 퍼스트 = 팀관리앱에서 통한 유일 성장 템플릿) + 활성화 지표(콜드 인스타보다 크리에이터 추천이 활성화율 높음).
> ⚠️ 아래 구독자 수·연락처는 웹 검색 스니펫 기반 **추정치**. 실제 집행 전 각 채널 유튜브 "정보" 탭에서 비즈니스 이메일·최신 구독자 직접 확인 필수.

## 0. 리서치 메타 결론 (기대치 세팅)

**조기축구 "팀 운영" 니치 유튜브 지형은 생각보다 작다.**

- **대형 채널(수십만~백만)은 우리 타겟이 아님**: 감스트(248만)·제이풋볼(156만)·슛포러브(152만)·이스타TV·달수네라이브(박문성) = 전부 **예능·해외축구·해설·프리스타일**. 시청자가 "팀 운영 도구"를 찾는 사람이 아님. 비싸고 정합성 낮음 → 후순위.
- **우리 fit = 마이크로/미드 "아마추어 팀 운영·조기축구 라이프" 채널**: 실제로 팀 굴리는 회장·감독이 하는 채널. 볼륨은 작지만 시청자 = 정확히 회장·총무·선수.
- 시사점: **"큰 채널 하나 유료 PPL" 아니라 "맞는 마이크로 채널 여러 개 + 성과형 딜"** 게임. 초기 현금 부담 최소로 시작 가능.

## 2. 채널별 접근 전략 (현금 프레임 → "유용한 도구" 프레임)

우리 앱은 **무료**라 "제품 협찬"이 약함. 대신 아래 3구조로:

### A. 무료~저비용 자연노출 (마이크로 채널 다수)
- 제안: "당신 팀(동고FC·한마음FC 등) 운영에 직접 써보고, 좋으면 브이로그에 자연스럽게 노출."
- 대가: 그 채널 팀에 **프리미엄/AI 기능 무제한 무료** + 제휴 코드. 현금 0~소액.
- 왜 통하나: 이들은 이미 매주 출석·회비·라인업으로 고생 중 = 진짜 pain. 실사용 후기가 광고보다 강함.

### B. 성과형 제휴 (CPA) ⭐ 비용 리스크 해법
- 제안: 전용 초대/제휴 링크 → **가입당 or 팀 생성당 정액 지급**.
- 왜 우리한테 최적: 아직 활성화 깔때기가 새는 상태(활성화 25%)라 **고정비 PPL은 위험**. CPA는 결과 없으면 지출 0 → 리스크를 크리에이터와 분담. 큰 채널에도 무료 리스크로 접근 가능.
- 단가 예시(협상): 가입 1건 X원 / 팀 생성 1건 Y원(가입보다 높게). 상한 캡 설정.

### C. 앵커 1채널 유료 테스트
- 정합성 최상(동고FC 등) 딱 하나에만 소액 전용 협찬 영상 → ROI 측정 후 확대 판단.
- 이때만 고정비 지출. 나머지는 A·B로.

저도 5년째 조기축구 회장이라 매주 출석 투표·회비 대조·라인업 짜는 게

거창한 광고 요청은 아니고, {채널명}님 팀에 먼저 그냥 써보시라고요.

## 4. 실행 순서 & 예산 시나리오

1. **채널 리스트 확정**(사용자+나): 위 후보 + 수도권 조기축구 브이로그 마이크로 5~10개 발굴 → 각 채널 정보 탭에서 **비즈니스 이메일·구독자·최근 업로드일** 기록.
2. **A(무료 협찬) 메일 5~10곳 동시 발송**: 현금 0. 회신율만 봐도 채널 반응 데이터.
3. **B(성과형) 제안**: A에서 반응 좋은 채널에 CPA 딜.
4. **C(앵커 유료)**: 정합성 최상 1곳 소액 테스트.
5. 예산: **1단계 0원 → 2~3단계도 CPA라 성과 없으면 0원 → 앵커 유료 1건만 소액(예: 소정)**. 즉 초기 현금 거의 없이 시작.
