# 수동 작업 필요 항목

코드로 해결 불가하거나 데이터/에셋이 필요한 항목들.

## 1. OG 이미지 제작 (1.9)
- **필요**: 1200x630px 브랜드 이미지
- **용도**: 카카오톡/SNS 공유 시 링크 미리보기 이미지
- **내용 추천**: PitchMaster 로고 + 앱 스크린샷 + "조기축구 팀 관리" 텍스트
- **적용 위치**: `src/app/layout.tsx` openGraph.images에 추가
- **방법**: Figma/Canva로 제작 후 `public/og-image.png`에 저장

## 2. 소셜프루프 수치 업데이트 (1.7)
- **현재**: "100+ 관리된 경기", "1,500+ 참석 투표", "37 팀 멤버", "4개월+"
- **문제**: 하드코딩 + 수치가 작아서 오히려 역효과 가능
- **옵션 A**: 실제 DB에서 동적으로 조회 (서버 컴포넌트로 전환 필요)
- **옵션 B**: 최신 수치로 업데이트 (FK Rebirth 기준: 62+경기, 4팀, 90+멤버)
- **옵션 C**: 스탯 섹션 자체를 제거하고 Before/After에 집중
- **적용 위치**: `src/app/login/page.tsx` 93-104행

## 3. 사용자 후기 강화 (2.1)
