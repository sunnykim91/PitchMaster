```

────────────────────────────
합계              29,400원/월
```

## 5. 가격 정책 제안

### 3-Tier 구조

```
Free (영구 무료)
├─ 15명 이하 팀
├─ 월 2경기 제한
├─ 기본 기능 (투표 · 회비)
└─ AI 기능 없음

Pro — 9,900원/월 (또는 99,000원/년, 17% 할인)   ← 핵심 전환 티어
├─ 인원 제한 없음
├─ 경기 무제한
├─ 전술판 · 기록 · 랭킹 · 게시판 전부
├─ AI 시그니처 카피
├─ AI 경기 후기
└─ OCR 자동 회비 처리

Premium — 14,900원/월 (또는 149,000원/년)   ← 상위 플랜
├─ Pro 전부 포함
├─ AI 라인업 추천 + 설명
├─ 자연어 챗봇 (미납자 조회 등)
├─ 시즌 어워드 프리미엄 카드
└─ 우선 지원
```

### 가격 책정 근거

**1인당 환산** (팀 평균 13.3명 기준)
- 9,900원/팀 = **744원/인/월**
- 14,900원/팀 = 1,128원/인/월
→ 1인이 내는 월 회비의 2~3% 수준

**팀 총 운영비 대비**
- 축구 동호회 월 운영비 평균 40~100만원 (구장 + 장비 + 회식)
- 월 15,000원은 **전체 지출의 1.5~2%**
→ 의사결정 부담 거의 없음

**SaaS 시장 벤치마크**
- Notion Team $10/인 → 10인 팀 기준 140,000원/월
- Slack Pro $8.25/인 → 10인 팀 기준 115,000원/월
- TeamSnap (미국 팀 관리) $10~25/월/팀
- SportEasy (유럽) €4.99~9.99/월/팀 = 7,000~14,000원
→ PitchMaster 9,900~15,000원은 **정확히 글로벌 경쟁 가격대 중간**

### 개발비(Claude Code Max) 회수 — 선택적 목표

Max는 본업에서도 쓰는 공유 자산이므로 **PitchMaster가 반드시 회수할 의무는 없음**.
다만 사이드 프로젝트로 얼마나 회수 가능한지 참고용:

- 보수적 전환 (126,145원) → Max 월 140,000원의 90% 커버
- 현실적 전환 (200,750원) → Max 완전 커버 + 잉여 60,000원
- 공격적 전환 (275,355원) → Max 회수 + 확실한 흑자

→ 목표가 된다면 **Pro 전환율 40%+ 가 Max 회수 분기점**.
