## HIGH — 사용성에 직접 영향

- [ ] 골 삭제 확인 다이얼로그 — 현재 확인 없이 즉시 삭제됨
- [ ] "전원 참석 처리" 확인 다이얼로그 — 실수 탭 시 전원 출석 처리
- [ ] 경기 목록 빈 상태 개선 — 일정 등록 버튼이 스크롤 밖에 있음, 인라인 액션 추가
- [ ] 게시판 공지 고정(PIN) 기능 — 회장이 중요 공지를 상단에 고정
- [ ] 회원 벌크 사전등록 — 25명 하나씩 등록은 고통, CSV 또는 다중 입력 폼

## MEDIUM — 편의성 개선

- [ ] 납부현황 "미납만 보기" 필터 — 운영진이 미납자만 빠르게 확인
- [ ] 경기 상세 "미투표자" 필터/하이라이트 — 누가 아직 투표 안 했는지 한눈에
- [ ] 팀 로고 파일 업로드 — URL 입력 대신 이미지 업로드
- [ ] 랜딩 페이지 소셜프루프 하드코딩 제거 — 수치 오래되면 역효과
- [ ] 온보딩 생년월일 기본값 개선 — 1998 → 빈 값 또는 1990
- [ ] OCR 500K 임계값 경고 — 50만원 이상 거래 무시됨, 경고 표시 필요
- [ ] 벌금 관리 별도 페이지 — docs/penalty-feature-backup.md 참고

## LOW — 완성도

- [ ] 게시판 페이지네이션 — 글 많아지면 무한 스크롤 필요
- [ ] 개별 선수 상세 통계 페이지 — 랭킹에서 클릭 → 상세
- [ ] 팀 규정 순서 변경 — 드래그앤드롭 또는 순서 번호
- [ ] 알림 설정 페이지 — 투표/경기 리마인더 on/off
- [ ] 사이드바 항목 그룹핑 — "팀 관리" vs "개인" 구분
- [ ] 유니폼 프리뷰 크기 확대 — 모바일에서 16x16 너무 작음
- [ ] 스포츠 유형 변경 기능 — 팀 생성 후 축구↔풋살 전환 불가
