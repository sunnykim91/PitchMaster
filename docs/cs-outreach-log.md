## 신규 가입 팀 발송 가이드

새로 가입하는 팀이 있으면 아래 절차로 발송:

1. DB에서 신규 팀 확인 (admin 페이지 또는 쿼리)
2. 회장 번호로 카톡 친구 추가 시도
3. 카톡 가능 → 위 공통 메시지에 {팀명}, {회장명} 치환 후 발송
4. 카톡 불가 → 인앱 알림으로 대체 발송
5. 이 문서에 기록 추가
