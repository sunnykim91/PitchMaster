## 본문 초안

경기 전날이면 늘 같은 고민이 반복됩니다. *오늘 몇 명 오지? 누구를 어느 자리에 두지? 쿼터마다 누가 쉬고 누가 뛰지?* `[★실데이터: 본인 팀 규모·종목 한 줄 — 예: 풋살 6:6, 매주 12~16명]` 출석을 받아놓고도 막상 **라인업을 짜는 데** 또 시간이 듭니다. 종이에 적고, 지우고, 한 명 빠진 거 뒤늦게 발견하고… 이 글은 축구·풋살 **라인업과 포지션을 손으로 짤 때 매번 막히는 지점**과, 그걸 무료 앱으로 자동화하는 법을 정리했습니다.

### 라인업을 손으로 짜면 매번 막히는 4가지

1. **누가 오는지부터 흐릿하다** — 단톡방 투표 결과를 보고 머릿속으로 인원을 세는 단계. 막판 변동(불참·추가)이 생기면 다시 처음부터.
2. **포지션이 겹치거나 빈다** — 공격수만 다섯, 수비는 한 명. 종이로 짜면 전체 균형이 한눈에 안 들어옵니다. 특히 **풋살은 6명(5+GK 또는 6명 필드)이라 한 자리만 어긋나도 티가 큽니다.**
3. **쿼터 교대가 누락된다** — "이번 쿼터 누구 쉬었지?"가 헷갈려서 특정 인원만 계속 뛰거나, 어떤 사람은 두 쿼터 내내 벤치. 출전 시간 불공평이 분쟁으로 번집니다.
4. **포지션별 역할 설명이 안 된다** — 처음 그 자리를 맡은 사람은 "여기서 뭘 해야 하죠?"를 묻는데, 매번 말로 설명하기 번거롭습니다.

### 라인업 짜는 기본 4단계 (수동 기준)

자동화 전에, 라인업을 제대로 짜는 순서를 정리하면 이렇습니다.

1. **출석 확정** — 오는 사람 명단 고정 (막판 변동 반영)
2. **포메이션 선택** — 축구는 4-4-2·4-3-3 등, 풋살은 인원에 맞춰 5인제/6인제 포메이션
3. **포지션 배치** — 선호 포지션·실력을 고려해 각 자리에 배정, 좌우/공수 균형 확인
4. **쿼터 교대 안배** — 쿼터마다 출전·휴식을 돌려서 출전 시간 균등하게

손으로 하면 3·4번에서 매번 시간이 녹습니다.

### 무료 앱으로 자동으로 짜는 법

PitchMaster(무료)는 위 3·4단계를 자동으로 처리합니다.

- **출석자만 고르면 자동 배치** — 참석 투표에서 "참석"한 인원을 그대로 가져와, 선택한 포메이션의 각 포지션에 자동으로 채웁니다. *축구 포지션 짜는 앱*을 찾으셨다면 이 자동 편성이 핵심입니다.
- **실력·포지션 균형 분배** — 라인(공격/미드/수비)별로 인원을 고르게 나누고, 선호 포지션을 우선 반영해 한쪽에 쏠리지 않게 배치합니다.
- **쿼터별 자동 교대(로테이션)** — 쿼터 수를 정하면 출전·휴식을 자동으로 돌려, 특정 인원만 계속 뛰거나 쉬는 일을 막습니다. **풋살은 키퍼·교대 순번 룰렛**으로 골키퍼와 교대 순서까지 자동으로 정할 수 있습니다.
- **전술판에서 손으로 미세 조정** — 자동 배치 결과를 드래그로 옮겨 원하는 대로 수정. 쿼터마다 다른 포메이션도 가능하고, 홈/어웨이 유니폼 색까지 반영됩니다. 짠 결과는 그대로 공유·캡처해서 단톡방에 올릴 수 있습니다.

`[★실데이터: 자동 편성 한 번 돌려본 소감 1~2문장 — 예: "손으로 20분 걸리던 게 한 번 누르니 끝"]`

### 풋살 포지션·축구 포지션, 역할까지 자동 안내

"**풋살 6명 포지션**"이나 "**풋살 중앙수비수 역할**"처럼 포지션별로 뭘 해야 하는지 막막할 때가 많습니다. PitchMaster는 라인업을 짜면 **각 포지션의 역할·주의점**을 쿼터별로 보여줍니다.

- **축구 11인제 + 풋살 5·6인제** 포지션 역할 가이드 제공
- 본인이 맡은 자리의 핵심 임무와 연계 플레이를 카드로 안내 → 처음 그 자리를 맡은 사람도 바로 이해
- 쿼터마다 포지션이 바뀌면 그 쿼터 기준으로 다시 안내

말로 매번 설명할 필요 없이, 각자 자기 포지션 역할을 앱에서 확인하면 됩니다.

### 축구(11인제) vs 풋살, 뭐가 다른가

- **축구 11인제**는 4-4-2·4-3-3 같은 정식 포메이션 + 11자리 배치가 핵심.
- **풋살은 6:6(필드 5 + GK)** 기준이 표준이라 인원이 적은 만큼 한 자리 균형이 더 중요하고, **쿼터·키퍼 교대**가 운영의 핵심입니다.
- 두 종목 모두 같은 앱에서 종목만 선택하면 그에 맞는 포메이션·포지션·역할이 적용됩니다.

### 무료로 쓸 수 있나요?

네. 라인업 자동 편성·전술판·포지션 역할 가이드 모두 **무료**로 쓸 수 있습니다. "*무료 포지션 앱*", "*축구 포메이션 앱 추천*"을 찾으셨다면, 설치 없이 웹에서 바로 시작할 수 있고 앱(안드로이드)도 있습니다.
