## 경쟁사(축구고) 대비 포지셔닝

| | 축구고 | PitchMaster |
|---|---|---|
| 강점 | GPS·Wear OS·OVR 능력치 (선수 개인 성장) | 회비·AI·공지·전술판 (팀 운영자) |
| **기기 접근성** | 폰 전용 (Wear OS 보조) | **PC·태블릿·폰 모두 — 설치 없이 브라우저** |
| 약점 | 회비·공지·AI·PC 전무 | 네이티브 아닌 PWA |
| 규모 | 50+ 다운로드 (초기) | 82팀 647명 실사용 |

**PC 접근성은 숨은 차별점**: 감독·총무는 경기 전날 PC로 준비, 현장은 모바일 — 이중 사용 패턴을 지원. 전술판 11명 드래그·회비 엑셀 업로드·OCR 처리는 큰 화면이 압도적으로 효율적. 축구고는 네이티브라 PC 접근 자체가 불가.

**포지셔닝 카피 후보**:
- "선수 개인은 축구고, 팀 운영은 PitchMaster"
- "**경기 전날 밤엔 PC, 현장엔 폰**"
- "11명 드래그, 폰은 좁다 — 넓은 화면에서 전술판 짜세요"
- "설치 없이 브라우저에서 바로"
- 직접 대비 없이 **"조기축구 총무 구원 앱"** 단독 메시지로 밀어도 좋음
