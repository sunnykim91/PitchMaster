## 일반 전술판 앱이 부족한 이유

검색 상위에 노출되는 풋살 전술판 앱 대부분이 그리기 도구에 머물러요. 코치·분석가가 전술 회의용으로 쓰기엔 충분한데, 매주 회원 17명·20명 출석 받아서 라인업 짜야 하는 운영자한테는 한참 모자랍니다.

제가 매주 마주하는 작업이 이런 식이에요. 출석 투표 결과 보면서 18명 중 16명 참석, 그중 골키퍼 가능자 2명, 좌측 수비 선호 3명을 머릿속에서 매칭. 종이에 그려놓고 단톡방에 사진 공유. 다음 주 또 처음부터.

그림 도구로 한 번 켜는 데 30분 사라지는데, 이거 매주 반복이에요. 그래서 결국 도구 만들어서 쓰기 시작했고요.

## 1. 회원 데이터 연동 — 익명 X·O 말고 실제 이름

등록된 회원 목록이 전술판에 그대로 뜨고, 출석 투표한 사람만 자동으로 후보군에 올라오는 구조예요.

김민준·박서준 이름이 코트 위에 보여야 단톡방 공유했을 때 "내 자리 어디야?" 질문이 안 나옵니다.

그림 도구에서 매번 회원 이름 라벨링하면 20명짜리 라인업 한 번 만드는 데 10분 걸려요. 회원 등록 한 번으로 끝나는 게 정답입니다.

## 2. 자동 편성 — 출석 인원·선호 포지션 종합

출석 16명·골키퍼 2명·좌측 수비 선호 3명 같은 데이터를 바탕으로 균형 잡힌 팀을 자동으로 나눠주는 기능이에요. 운영자가 머릿속에서 외워야 할 데이터가 0이 됩니다.

고급형 도구는 회원별 과거 출전 기록·MVP 횟수까지 종합해서 실력 균형까지 맞춰줘요. 매주 다른 조합을 자동 추천해주니까 "지난주랑 똑같은 팀이네"라는 불만도 안 나오고요.

## 3. 풋살 6인제 표준 포메이션 + 인원별 대응

한국 조기 풋살은 6:6(필드 5 + 골키퍼 1)이 표준이에요. FIFA 공식 5:5보다 6:6이 동호회에서는 사실상 표준입니다.

6인제 포메이션은 3가지가 기본이에요.

2-2-1 — 균형형. 수비 안정 + 공격 1명 고정
1-3-1 — 미드 강화. 중원 장악·점유율 위주
2-1-2 — 공격형. 수비 2 + 공격 2, 미드 1명만

이 3가지는 기본 제공되어 있어야 하고, 인원이 적게 모이는 날(3~4인제)도 따로 포메이션이 준비되어 있어야 갑작스러운 결원에 대응 가능합니다.

## 4. 자체전 모드 — A vs B 동시 편성 + 통계 분리

풋살 동호회는 정기 경기보다 자체전 비중이 커요. 인원 많을 때 3파전·4파전 돌리는 거요.

A팀과 B팀을 같이 편성하면서도 시즌 통계에는 합산 안 되게 분리되는 모드가 필요합니다. 자체전 골 기록이 정기 경기 기록이랑 섞이면 시즌 득점왕이 누군지 헷갈리거든요.

일반 전술판은 한 팀만 그릴 수 있어서 자체전 때 화면 두 번 띄워야 해요. A팀 편성하고 저장, 다시 B팀 편성. 그러는 동안 회원 정보 또 입력. 자체전 모드 있으면 한 화면에서 양 팀 동시 편성 + 코트 위에 같이 표시되니까 편하더라고요.

## 5. GIF·영상 공유 — 단톡방 즉시 전송

편성한 전술을 단톡방에 공유할 때 정적 이미지보다 움직이는 GIF가 직관적이에요. 1쿼터 포메이션, 2쿼터 변형, 3쿼터 복귀 같은 흐름을 GIF 한 장으로 보여주면 사전 미팅 없이도 회원들이 본인 포지션을 이해합니다.

영상 편집 도구 별도로 안 거치고 앱 안에서 바로 출력되는 게 핵심이에요. 운영자가 영상편집까지 배우라는 건 무리니까요.

## PitchMaster 풋살 전술판 — 무료 사용 후기

위 5가지 조건을 모두 충족하는 무료 도구가 PitchMaster예요. 직접 만들어 쓰다가 다른 팀들도 쓸 수 있게 공개한 건데, 현재 100팀 800여 명이 무료로 사용 중입니다.

회원 등록 한 번으로 전 경기에 자동 매칭되고, AI 자동 편성이 선호 포지션·과거 출전·MVP를 종합해서 균형 분배를 추천해줍니다. 풋살 3~8인제 16종 포메이션 + 쿼터별 다른 포메이션 설정 가능하고요. 자체전 모드에서 A팀·B팀 동시 편성하면 시즌 통계와 분리되어 저장됩니다. 전술 시퀀스는 GIF로 자동 출력되어 단톡방에 바로 공유 가능해요.

축구 11인제도 같은 앱에서 같이 운영 가능합니다. 회비 관리·출석 투표·MVP 투표까지 한 곳에서 처리되니까 운영자가 외울 앱이 줄어들어서 편하더라고요.

## 정리 — 풋살 전술판 앱 고르실 때 체크

회원 데이터 연동되는가, 출석 인원 기반 자동 편성 있는가, 풋살 6인제 포메이션(2-2-1, 1-3-1, 2-1-2) 기본 제공하는가, 자체전 모드 + 시즌 통계 분리되는가, GIF·영상 공유가 앱 내에서 바로 되는가.

이 5가지 중에 3개 이상 빠지면 그 앱은 "그림 도구"라고 보시면 됩니다. 풋살 동호회 운영용은 아니에요.

그림 도구는 코치·분석가에게 맡기고, 운영자는 자동화된 도구로 시간 아끼시는 걸 추천드립니다. 매주 30분 사라지는 거 1년이면 26시간이에요.
