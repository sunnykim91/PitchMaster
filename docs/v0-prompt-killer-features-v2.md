## 🎯 3종 공통 — 반드시 추가할 5가지

### 1. 희소성·등급감 (Rarity)
숫자 OVR만으로는 부족함. **카드 자체가 "이 카드는 특별하다"고 외쳐야 함.**

- **홀로그래픽 포일 효과** — Gold(90+) 카드는 무지개 그라디언트가 카드 표면에 흐르는 느낌. CSS `conic-gradient` + `mix-blend-mode: overlay` + 노이즈 텍스처. 정적이어도 OK.
- **빛 산란 / 광택 streak** — 카드 좌상단에서 우하단으로 흐르는 대각선 하이라이트. 살짝 blur.
- **rarity별 외곽 효과**:
  - **ICON (90+)**: 두꺼운 골드 메탈릭 테두리 + 외곽 골드 글로우(box-shadow `0 0 60px gold/30%`) + 카드 뒷배경에 발광하는 후광
  - **HERO (80–89)**: 오렌지/코랄 메탈릭 테두리 + 코랄 글로우
  - **RARE (70–79)**: 청록 holographic + teal 글로우
  - **COMMON (45–69)**: 깔끔한 무광 단색 (대비를 위해 일부러 심심하게)
- **rarity 라벨**을 카드 상단 또는 외곽에 작게 — `ICON` / `HERO` / `RARE` / `COMMON` (영문 letter-spacing 넓게)
- **Sparkle/particle 점**들을 카드 배경에 미세하게 (특히 ICON 등급)

### 2. 스토리·맥락 라벨 (Story Tags)
숫자 옆에 **반드시 한 줄 맥락**을 붙여라. 사람은 "28골"이 대단한지 모름. "팀 1위 / 역대 2위 / 13경기 연속" 이라고 적어주면 그제서야 자랑이 됨.

- 모든 핵심 스탯 옆에 작은 컨텍스트 chip:
  - `골 28` ← `🏆 팀 1위` 또는 `📈 시즌 최다`
  - `MOM 15` ← `⭐ 역대 1위`
  - `출석률 95%` ← `🔥 17경기 연속`
  - `승률 67%` ← `상위 10%`
- mock 데이터에 이런 라벨 필드 추가:
  ```ts
  type StatWithContext = {
    label: string;
    value: string;
    rank?: string;        // "팀 1위", "상위 5%", "역대 2위"
    streak?: string;      // "13경기 연속", "5경기 연속 득점"
    badge?: "fire" | "trophy" | "rocket" | "crown";
  };
  ```
- 컨텍스트가 있을 때만 chip 표시. 없으면 숫자만.

### 3. 한 장 안의 위계 (Visual Hierarchy)
지금은 모든 요소가 비슷한 무게라 눈이 머물 곳이 없음. **카드 안에서도 "주연·조연·엑스트라"가 명확해야** 함.

- 주연 1개(OVR/주인공 이름) — 압도적 크기, 글로우, 가장 밝게
- 조연 2~3개(핵심 스탯 1~2개 + 포지션) — 중간 크기, 강조 색
- 엑스트라(기타 정보) — 작게, opacity 60% 이하
- "한 화면을 캡쳐했을 때 0.3초 안에 주인공이 누구인지" 보이게.

### 4. 모션 없이도 살아있게 (Implied Motion)
v0는 정적 이미지지만, **"움직이고 있다"는 인상**을 줘야 함.

- 카드 뒤에 흐릿한 motion blur 그라디언트(주인공이 달려나간 자국처럼)
- 빛이 한쪽에서 들어오는 dramatic lighting
- 약간의 vignette (외곽이 어두워짐) — 시선을 중앙으로 모음
- 배경에 매우 흐릿한 축구장/스타디움 라인 패턴 (5% opacity)

### 5. 공유 가능한 "한 장 모드" (Share-Ready)
화면에서 보는 것과 별개로, **공유 버튼 누르면 1080×1920 인스타 스토리 비율 / 1080×1080 정사각형으로 정확히 들어가는 별도 레이아웃**을 같이 디자인.

- 인스타 스토리 모드: 위 1/3은 카드, 가운데는 핵심 스탯 3개 + 스토리 라벨, 아래 1/3은 팀명·시즌·`pitch-master.app` CTA
- 카톡 미리보기 모드: 1200×630 OG 카드 — 좌측에 카드, 우측에 이름·팀·핵심 수치 3개
- "공유하기" 버튼 누르면 이 share-ready 레이아웃이 모달로 떠야 함

## 1️⃣ 선수 카드 v2 — "갖고 싶은 한 장"

### 지금 결과물의 문제
- 평면적인 그라디언트 박스에 stat이 박혀 있음
- placeholder 아바타가 카드 면적의 25%를 비워둠
- OVR 92인데 92라는 숫자가 그저 노란색일 뿐, "92는 전설"이라는 신호가 없음
- 모든 카드가 같은 레이아웃이라 등급 차이가 안 느껴짐

### 새 요구사항

**A. ICON 카드 (90+) 전용 디자인 강화**
- 카드 배경: 골드 ↔ 다크 ↔ 골드 conic-gradient + 노이즈
- 카드 외곽: 두꺼운 골드 메탈릭 프레임(double border, 안쪽은 어둡고 바깥은 빛남)
- OVR 숫자: 거대한 골드 (text-shadow로 글로우 + 미세한 emboss)
- 카드 뒤 후광: `radial-gradient` 골드 글로우가 카드 뒤로 퍼짐
- 카드 상단: `★ ICON ★` 또는 `LEGENDARY` 라벨 (작게, letter-spacing 넓게)
- 미세한 sparkle dot 5~10개를 카드 면 위에 흩뿌림

**B. 선수 일러스트 자리를 똑똑하게 처리**
- 사진이 없을 때 placeholder 대신:
  - **거대한 등번호 #7** 을 카드 중앙에 반투명으로 배치 (300px+, opacity 15%)
  - 그 위에 선수 이름이 올라옴
  - 또는 추상적인 축구공/궤적 SVG 그래픽
- 사진이 있으면 자연스럽게 swap 가능하도록 컴포넌트 prop으로 `photoUrl?: string`

**C. 스탯 영역 위계 개편**
- 6개 스탯을 동일 그리드에 박지 말고:
  - **핵심 1개**(포지션의 시그니처 스탯, FW면 골)을 큰 카드로 — 숫자 48px+
  - 그 옆에 컨텍스트 chip("🏆 팀 1위")
  - 나머지 5개는 작은 그리드로
- 핵심 스탯은 primary 색으로 강조

**D. 카드 하단 "시그니처" 영역**
- 선수 이름 아래에 한 줄 캐치프레이즈:
  - `"15골 8어시 — 시즌 MVP 후보"`
  - `"22경기 출장 · 클린시트 7회"`
- 이건 백엔드에서 자동 산출 가능 (가장 인상적인 스탯 2개를 골라 문장 생성)

**E. 카드 뒷면(Flip)**
- 카드 클릭 시 3D flip 애니메이션 (CSS `transform: rotateY(180deg)`)
- 뒷면에는: 시즌 모든 스탯 표 + 최근 5경기 결과 + "이 카드 공유하기"
- 모바일에서도 동작하게

### 변형 4종을 한 페이지에 나열
4개 등급 카드를 한 페이지에 나란히 보여줘서 **차이가 한눈에 보이게**. 지금처럼 셀렉터로 1장만 보여주지 말고, 4장이 동시에 보여야 등급 시스템이 매력적임을 깨닫게 됨.

## 2️⃣ 시즌 어워드 v2 — "트로피 시상식"

### 지금 결과물의 문제
- 7개 카드가 동일 크기로 나열되어 시상식이 아니라 그냥 리스트
- 트로피 아이콘이 작아서 시상감이 안 남
- 시즌 전적 카드가 밋밋함 (60% 승률이 대단한지 모름)
- "이 시즌의 주인공"이 없음

### 새 요구사항

**A. 페이지 최상단 "시즌 인트로" 시퀀스**
- 화면 첫 1뷰는 거의 영화 타이틀처럼:
  ```
  PITCHMASTER PRESENTS

  2026 시즌
  FC 피치마스터

  THE SEASON
  AWARDS
  ```
- 거대한 타이포그래피, letter-spacing, 어두운 배경 + 코랄 액센트
- 작은 텍스트로 "총 20경기 · 12승 3무 5패 · 승률 60%"
- 살짝 vignette + 골드 particle

**B. "시즌 MVP" 슈퍼 카드 (가장 위에 1개)**
- 7개 어워드 중 **가장 임팩트 있는 1명을 시즌 MVP로 자동 선정** (득점왕 또는 최다 MOM)
- 이 사람만 거대한 카드(전체 폭, 높이 320px+)로:
  - 좌측: 선수 카드 1️⃣ ICON 등급의 컴팩트 버전
  - 우측: `2026 시즌 MVP` 라벨 + 이름(48px+) + 시그니처 스탯 3개 + 한 줄 캐치프레이즈
  - 배경: 골드 글로우 + 후광
- 이 카드가 "시즌의 주인공" 역할

**C. 나머지 6개 어워드는 2×3 또는 1×6 갤러리**
- 각 어워드를 작은 트로피 카드로 (모바일은 2×3, 데스크탑은 3×2)
- 각 카드:
  - 상단: 거대한 트로피/메달 SVG 일러스트 (아이콘이 아니라 일러스트 — 50px+, 시상별 다른 디자인)
  - 라벨: `득점왕` (작게, gold 컬러)
  - 수상자: 이름(20px, bold)
  - 수치: 큰 숫자 + 단위
  - 컨텍스트 chip: "전 시즌 대비 +5골" 같은 변화량
- 카드 호버 시 살짝 lift + 트로피가 살짝 회전(transform)

**D. 시즌 전적 카드를 "성과 리캡"으로 재구성**
- 단순 W/D/L bar 대신:
  - `12W` `3D` `5L` 를 큰 숫자로
  - 가장 큰 승리: `vs 강남 FC 7:1` (베스트매치를 여기에 흡수)
  - 가장 많은 득점: `김민수 15골`
  - 가장 많이 본 결과: `78%가 승리/무승부` 식
  - 한 줄 요약: `"우승 후보다운 시즌이었다"` 자동 생성

**E. 페이지 하단 "이 시즌을 하나의 이미지로"**
- 7개 시상 + 전적 + MVP 를 모두 담은 1080×1920 인스타 스토리 카드 미리보기
- "스토리로 공유" 버튼 → Web Share API
- "이미지 저장" → SVG/PNG 다운로드
- 카톡으로 공유했을 때 OG 카드도 같은 무드여야 함

## 3️⃣ 커리어 프로필 v2 — "나의 시즌 리캡"

### 지금 결과물의 문제
- 깔끔하지만 평범 — 그냥 잘 만든 프로필 페이지
- 시즌 안에 어떤 드라마가 있었는지 안 보임
- 최근 경기가 단순 리스트라 하이라이트가 없음
- "이 사람을 친구에게 소개하고 싶다"는 마음이 안 듦

### 새 요구사항

**A. 히어로 섹션을 "영화 인트로"로**
- 화면 첫 1뷰 가득 (거의 100vh)
- 배경: 팀 컬러 그라디언트 + 흐릿한 스타디움 라이트 + 노이즈
- 중앙: 거대한 선수 이름 (clamp(48px, 12vw, 96px), 한글 자간 넓게)
- 그 아래 한 줄: `"FW · 2026 시즌 · 18경기 출장"` 작게
- 그 아래 **시그니처 한 문장** (자동 생성):
  - `"15골 8어시 — 팀 득점왕"`
  - `"5경기 연속 MOM에 빛난 스트라이커"`
  - `"승률 82%, 그가 뛰면 팀은 이긴다"`
- 가장 아래: 작은 PitchMaster 워터마크 + 살짝 아래로 향한 화살표(scroll cue)

**B. 시즌 다이제스트를 "소유감 있는 카드 영역"으로**
- 선수 카드(1️⃣)를 페이지 안에 **실제 사이즈로** 박아넣기 (compact 버전 말고)
- 카드 옆/아래에 핵심 4지표를 거대한 숫자로:
  - `15` 골
  - `8` 어시
  - `82%` 출석률
  - `67%` 승률
- 각 숫자 옆에 컨텍스트 chip (`팀 1위`, `상위 10%`, `13경기 연속`)
- 이 영역이 페이지의 두 번째 핵심 hit

**C. "시즌 베스트 모먼트" 섹션 추가 (신규)**
- 자동 산출:
  - **베스트 경기**: 가장 많은 골/어시 한 경기 → "vs 강남 FC, 5:2 승, 본인 2골 1어시"
  - **시즌 첫 골**: 첫 골 경기 + 날짜
  - **연속 기록**: 가장 긴 연속 출장/연속 득점/연속 MOM
- 3개 카드로 가로 슬라이드 또는 세로 나열
- 각 카드: 큰 숫자 + 짧은 문구 + 날짜
- 인스타 reels 썸네일 같은 느낌 (16:9 또는 4:5)

**D. 최근 경기 타임라인 강화**
- 단순 리스트 ❌ → **세로 타임라인** 형태
- 좌측에 세로 라인, 각 경기마다 dot
- 큰 활약 경기(2골+ 또는 MOM)는 dot이 코랄로 강조 + 옆에 ⭐
- 각 경기 카드:
  - 결과 뱃지 (지금처럼)
  - 상대팀 + 스코어 (큰 글씨)
  - 본인 활약을 큰 이모지로 (⚽⚽ 🅰️ ⭐)
  - 우측 끝에 작은 화살표 → 클릭 시 경기 상세
- 평범한 경기는 작게, 활약 경기는 크게 (높이 다르게)

**E. 푸터 "내 카드 가져가세요"**
- 큰 코랄 CTA 카드:
  - "이 시즌의 내 카드를 다운로드"
  - "친구에게 공유" 
  - "우리 팀에도 PitchMaster"
- 작은 부제: "회원 가입 없이도 누구나 자기 시즌 카드를 만들 수 있습니다"

**F. 페이지 전체에 "공유" 버튼이 sticky**
- 모바일: 하단 fixed 버튼 `[ 내 카드 공유하기 ]` 코랄
- 누르면 share modal — 인스타 스토리/카톡/URL 복사
